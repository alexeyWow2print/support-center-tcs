# Настройка менеджеров

Данный раздел используется только в случае, если нужно, чтобы заказы и/или обращения с сайта распределялись между менеджерами по очереди.

Для этого отметьте галочкой необходимые пункты:

* **Распределять заказы автоматически**
* **Распределять обращения с сайта автоматически**

После чего нажмите **"Сохранить"**.

<figure><img src="../../.gitbook/assets/image (232).png" alt=""><figcaption></figcaption></figure>

### **Распределять заказы автоматически**

#### **Если параметр ВКЛючен**

При активации данного параметра, все заказы от новых клиентов будут автоматически распределяться между менеджерами. Каждый заказ будет закрепляться за менеджером в порядке очереди.\
Исключение могут быть, если у менеджера установлен [нерабочий день](polzovateli.md#ustanovka-nerabochikh-dnei-polzovatelya-sotrudnika), в этом случае очередь пропускается.

{% hint style="warning" %}
Распределение заказов происходит только для тех менеджеров, у которых в "[Роли и доступы](roli-i-dostupy.md)" включен соответствующий пункт: **"Возможность принимать заказ"**.

<img src="../../.gitbook/assets/image (218).png" alt="" data-size="original">
{% endhint %}

После того как заказ назначается менеджеру, за ним закрепляется клиент. Все дальнейшие заказы от данного клиента будут автоматически направляться закрепленному менеджеру.

{% hint style="info" %}
Автоматическое распределение применяется только к заказам от **новых клиентов**.
{% endhint %}

#### **Если параметр ВЫКЛючен**

При выключенном параметре, заказы от новых клиентов будут попадать в появившуюся вкладку **"Неразобранные"**. Любой менеджер может принять заказ, нажав кнопку _«Принять»_. Тот, кто сделает это первым, закрепит клиента за собой, и все будущие заказы от этого клиента будут поступать ему.

<figure><img src="../../.gitbook/assets/image (217).png" alt=""><figcaption></figcaption></figure>

### **Распределять обращения с сайта автоматически**

#### **Если параметр ВКЛючен**

При **включённом** параметре обращения равномерно распределяются между менеджерами (кроме тех, у кого отмечен [нерабочий день](polzovateli.md#ustanovka-nerabochikh-dnei-polzovatelya-sotrudnika)).

{% hint style="warning" %}
Распределение обращений происходит только для тех менеджеров, у которых в "[Роли и доступы](roli-i-dostupy.md)" включен соответствующий пункт: **"Возможность принимать обращения".**

<img src="../../.gitbook/assets/image (219).png" alt="" data-size="original">
{% endhint %}

#### **Если параметр ВЫКЛючен**

При активации данного параметра, все обращения попадают во вкладку _«Неразобранные»_. Менеджер может принять обращение, нажав кнопку _«Принять»_, после чего клиент закрепляется за ним. Все последующие обращения и заказы от этого клиента будут автоматически направляться принявшему менеджеру.
