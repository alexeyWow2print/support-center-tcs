# ВКонтакте

В этом руководстве мы расскажем, как подключить быструю авторизацию для клиентов вашего сайта через VK ID. Интеграция позволит пользователям входить на сайт типографии всего в один клик с помощью аккаунта ВКонтакте.

## Инструкция по созданию приложения VK ID

### Создание приложения

1. Откройте [страницу управления приложениями VK ID](https://vk.com/apps?act=manage).
2. Авторизуйтесь под аккаунтом, от имени которого будете управлять приложением. После входа вы попадёте на страницу «Мои приложения».
3. Нажмите кнопку **«Создать»**.

<figure><img src="../../../.gitbook/assets/Screenshot_001.png" alt=""><figcaption></figcaption></figure>

4. Вас перенаправит в документацию для разработчиков. На вкладке «Создание новых приложений» нажмите на ссылку **«Сайт»**.

<figure><img src="../../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

5. Сервис авторизации запросит доступ к вашим данным. Нажмите **«Разрешить»** или выберите нужный аккаунт.
6. Далее система запросит подтверждение через аккаунт. Нажмите **«Продолжить»**. В этом аккаунте будет создан бизнес-профиль VK ID (если он ещё не создан).

7. Теперь вы на странице «Мои приложения». Нажмите **«Создать»** (если приложений ещё нет) или **«Добавить приложение»** (если уже создано).

<figure><img src="../../../.gitbook/assets/Screenshot_3.png" alt=""><figcaption></figcaption></figure>

8. Откроется первый шаг создания — **«Регистрация приложения»**:

* Введите **название приложения** (оно будет отображаться пользователям в окне авторизации).
* Выберите тип платформы **«WEB»**.
* При необходимости загрузите изображение для иконки.
* Нажмите **«Далее»**.

<figure><img src="../../../.gitbook/assets/Screenshot_4.png" alt=""><figcaption></figcaption></figure>

### Настройка адресов и получение ключей

1. На втором шаге **«Данные для регистрации»** заполните поля:
   * **«Базовый домен»**: укажите домен вашего сайта **без** `https://` и без косой черты в конце. Например: `ваш-домен.ru`.
   * **«Доверенный Redirect URL»**: это адрес, на который VK ID вернёт пользователя после успешного входа.\
     Пример: `http://ваш-домен.ru/social/login/service/vkontakte`
2. Нажмите **«Создать приложение»**.

<figure><img src="../../../.gitbook/assets/Screenshot_11.png" alt=""><figcaption></figcaption></figure>

3. Для подтверждения создания на номер телефона, указанный в аккаунте VK ID, придёт SMS с шестизначным кодом. Введите его в открывшемся окне.

<figure><img src="../../../.gitbook/assets/Screenshot_6.png" alt=""><figcaption></figcaption></figure>

4. На следующем экране снимите все галочки предложенных быстрых входов и нажмите **«Настроить позже»**.

<figure><img src="../../../.gitbook/assets/Screenshot_12 (11).png" alt=""><figcaption></figcaption></figure>

5. Вас перенаправит в настройки приложения. Может появиться всплывающее окно **«Подтвердите профиль для управления приложением»** — нажмите **«Подтвердить»**.

**Важно:** Если ваш бизнес-профиль VK ID ещё не подтверждён, система перенаправит вас на страницу заполнения данных об организации (ИП или юрлице). **Без успешного подтверждения бизнес-профиля приложение не сможет работать, а авторизация будет недоступна.** После заполнения данных отправьте их на проверку и дождитесь её завершения .

<figure><img src="../../../.gitbook/assets/Screenshot_8.png" alt=""><figcaption></figcaption></figure>

6. Далее перейдите к разделу **«Доступы»** и добавьте к стандартным **Почту**. Эти доступы приложение будет запрашивать при авторизации пользователя на сайте.

<figure><img src="../../../.gitbook/assets/Screenshot_13 (9).png" alt=""><figcaption></figcaption></figure>

7. После этого вам будет предложено описать, для чего нужны дополнительные доступы. Укажите примерно следующее:

{% hint style="info" %}
Почта необходима для авторизации на сайте.
{% endhint %}

8. Далее просто подождите, пока пройдёт модерация приложения. Без её завершения авторизация работать не будет.

### Перенос ключей в платформу ТКС

1. В настройках созданного приложения найдите раздел **«Информация о приложении»**.
2. Скопируйте **«ID приложения»** (поле `Client ID`).
3. Скопируйте **«Защищённый ключ»** ( `Client Secret` ). Он нужен для выполнения запросов к API от имени приложения.

<figure><img src="../../../.gitbook/assets/Screenshot_9.png" alt=""><figcaption></figcaption></figure>

4. Перейдите в **Админ-панель** (в ТКС) → **Настройки** → **Другие настройки** → **Авторизация через соцсети**.
5. В строке напротив названия **«ВКонтакте»**:
   * В колонку **«Client id»** вставьте скопированный ID приложения.
   * В колонку **«Client Secret»** вставьте скопированный защищённый ключ.

<figure><img src="../../../.gitbook/assets/image (39).png" alt=""><figcaption></figcaption></figure>

### Завершение настройки

Что бы приложение начало функционировать, убедитесь, что ваш бизнес-аккаунт VK ID подтверждён. Без этого авторизация через созданное приложение работать не будет, а само приложение может быть через время заблокировано.

После завершения всех шагов авторизация через ВКонтакте станет доступна на сайте вашей типографии.
