# Настройки дизайна и загрузки макета

### **Настройки работы с макетом**

#### **Скачивание макета только после оплаты/гарантии оплаты**

При включении этой опции скачивание макета станет доступным только после полной оплаты заказа или установки гарантии оплаты.

#### **Изменение статуса макета товара в работу при выгрузке API**

Если вы используете выгрузку макетов через API, то при активации этой функции макет товара будет автоматически переводиться в статус **«Макет в работе».**

#### **Добавление белого листа на оборот**

Если в скачанном макете отсутствует оформление оборотной стороны, система автоматически добавит пустой белый лист.

#### **Добавление данных о заказе в макет конструктора**

Для макетов, созданных через конструктор: в углу изделия будут отображаться данные заказа (название сайта, номер заказа, номер товара и ФИО заказчика).

<figure><img src="../../../.gitbook/assets/image (3972).png" alt=""><figcaption></figcaption></figure>

#### **Автоматически принимать макеты**

При активации этой функции система будет автоматически принимать все загруженные макеты в заказах, без вашего участия.

### **Настройки отбраковки макета**

#### **Уведомить клиента**

При отбраковке макета товара вы можете отправить клиенту уведомление с указанием причины. Сообщение будет автоматически отправлено на email клиента, если данная функция активирована в разделе:\
**Настройки → Статусы → Уведомления**

#### **Сделать самим**

При выборе этого варианта доступны дополнительные опции:

* **«Выставлять счет»** – в списке услуг будут отображаться только позиции, созданные в разделе Справочника [«Доп. услуги»](../../../handbook/svoistva/untitled-4.md) и для которых в графе «Спец. услуга» указано значение "Создание / Редактирование макета".
* **«Не выставлять счёт»** – система не сформирует счет для клиента
