# Данные

В разделе Данные вы можете активировать интеграции со следующими базами данных:

### **DaData**

Интеграция с базой данных DaData уже встроена в систему. Вам необходимо только передвинуть ползунок для активации интеграции и подтвердить изменение статуса интеграции.

![](<../../.gitbook/assets/image (2376).png>)

{% hint style="info" %}
По ИНН из базы данных подтягивается только наименование и юридический адрес.
{% endhint %}

### **OpenDataBot**

Для интеграции с данным сервисом предварительно пройдите регистрацию на сайте [https://opendatabot.ua/api](https://opendatabot.ua/api)

После получения щелкните на названии "_OpenDataBot"_ интеграции в Настройки _→_ Интеграции _→_ Данные

![](<../../.gitbook/assets/image (1181).png>)

и в открывшемся поле введите ключ доступа.

![](<../../.gitbook/assets/image (622).png>)

После внесения данных, не забудьте активировать интеграцию при помощи переключателя

![](<../../.gitbook/assets/image (2148).png>)

### **DataGovUA**

Интеграция уже встроена в систему, для работы с ней, включите ползунок и нажмите "Изменить" для подтверждения изменения статуса интеграции.

![](<../../.gitbook/assets/image (3064).png>)

Применение:

После прохождения интеграции с базами данных, при оформлении заказа в Корзине, при вводе ИНН Компании Плательщика, информация о ней будет подтягиваться из интегрированного сервиса.

![](<../../.gitbook/assets/image (653).png>)
