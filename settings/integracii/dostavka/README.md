# Доставка

Вы можете пройти интеграцию с транспортными компаниями или добавить свой вариант в

> Настройки → Интеграции → Доставка

Чтобы добавить доставку в Интеграции→ Доставки нажмите <img src="../../../.gitbook/assets/Screenshot_1 (40).png" alt="" data-size="original">

![](<../../../.gitbook/assets/image (981).png>)

В списке возможных доставок, выберите нужную:

* eDost.ru (агрегатор доставок)
* [DPD](dpd.md)
* [CDEK](cdek.md)
* Нова Пошта
* DHL
* Top Delivery
* IML
* UPS
* Настраиваемая

{% hint style="danger" %}
**Надбавка**, устанавливаемая в [реквизитах компании](../../oplata/rekvizity.md#redaktirovanie-rekvizitov), распространяется исключительно на стоимость продукции и **не учитывается при формировании цены доставки.** Параметры налога (НДС) настраиваются в личном кабинете большинства сервисов доставки.
{% endhint %}

## Способы доставки, для которых требуется обращение к поставщику услуг.

Для интеграции с транспортными компаниями ([**DPD**](dpd.md)**,** [**Boxberry**](boxberry.md)**,** [**CDEK**](cdek.md)**, Нова Пошта, DHL, Top Delivery, IML, UPS**) и агрегатором доставок **eDost.ru**, предварительно обратитесь к ним для заключения договора и получения необходимых данных для интеграции (ключей доступа, accont, secure password, идентификатора магазина, api key и т.д.).

{% hint style="info" %}
**Внимание!** При заключении договора с транспортной компанией, обращайте внимание, чтобы договор был **для интеграции с онлайн-магазином**.
{% endhint %}

### **Добавление интеграции доставки на сайт**

Чтобы добавить интеграцию с транспортной компанией, выберите в списке нужную компанию и нажмите "Подключить"

![](<../../../.gitbook/assets/image (998).png>)

### Выбор параметров доставки

В открывшемся окне выполните следующие действия:

* Заполните _Название_
* Введите _данные_ для интеграции, полученные от транспортной компании
* Выберите город отправки из списка
* Установите _Скидку_ или _Наценку_ в рублях или процентах, если это требуется.
* Выберите _способы доставки_, предусмотренные интеграцией с транспортной компанией

![](<../../../.gitbook/assets/image (2025).png>)

После внесения всех данных, не забудьте их сохранить.

### Выбор способов доставки на сайте

После того, как интеграция с транспортной компанией пройдена,

> в Настройки → Доставки → Способы доставки →

отсортируйте доставки по _Типу_ и переключателями "Выкл./Вкл." активируйте те способы доставки, которые вы хотите отображать клиентам на сайте.

![](<../../../.gitbook/assets/image (2375).png>)

### Удаление интеграции на сайте

Вы можете включать и выключать интеграции для отображения на сайте при помощи переключателей напротив названия интеграции. Чтобы удалить интеграцию нажмите на иконку "Удалить".

![](<../../../.gitbook/assets/image (2024).png>)
