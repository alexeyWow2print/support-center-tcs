# Люверс

## Добавление Люверса

Чтобы добавить Люверс нажмите кнопку "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (2874).png>)

### Вкладка Описание

В открывшейся вкладке "Описание" заполните _Название_ и выберите:

* _Группу_ (при необходимости)
* Плавное распределение
* _Вес_ (в гр.)
* _Единицу измерения_: лист, м2, см.2, шт., компл., г/м2, пог. м., пог. см., пог. мм.
* Загрузите _картинку_ (минимальный размер 300x300px, jpg, gif, png, webp) и _иконку_ (минимальный размер 79x79px, jpg, gif, png, webp)
* _Описание_ (заполните поле)

Загруженные картинки и текст в поле _Описание_ будут отображаться на сайте, при наведении курсора мыши на параметр. Иконка ускорит поиск операции в папках или общем списке.

После внесения всех данных и загрузки изображений, нажмите "Сохранить".

После сохранения вкладки Описание, появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<../../.gitbook/assets/image (980).png>)

### Вкладка Фотогалерея

#### Добавление изображений

Чтобы добавить изображения в Фотогалерею нажмите кнопку "Добавить" → "Начать загрузку".

Требования к загружаемым файлам: минимальный размер 643x300, картинки jpg, gif, png, webp

![](<../../.gitbook/assets/image (2826).png>)

####

**Применение Фотогалереи**

Картинки загруженные во вкладку Фотогалереи отображаются в калькуляции продукта при выборе в Комплектующие→ Люверс → Галерея → Вкл. → Применить.

![](<../../.gitbook/assets/image (593).png>)

####

#### **Удаление изображений**

Чтобы удалить изображение нажмите кнопку "Удалить" ( <img src="../../.gitbook/assets/Screenshot_13.png" alt="" data-size="line">) напротив загруженного файла.

### Вкладка Свойства

{% hint style="info" %}
**Добавление свойства не является обязательным!** Если свойства не предусмотрены, пропустите вкладку и переходите сразу к ценам.
{% endhint %}

Чтобы добавить свойства (цвет, материал и т.д..) нажмите кнопку "Добавить".

![](<../../.gitbook/assets/image (2890).png>)

В открывшемся окне будут отображены свойства из Справочник → Свойства → Свойства комплектующих (вы можете добавить туда другие необходимые вам свойства: цвета, материал и т.д..).

![](<../../.gitbook/assets/image (1730).png>)

Выбранные свойства вы можете отображать на сайте, при выборе в Калькуляции продукта "Отображать свойства"

Пример в калькуляции:

![](<../../.gitbook/assets/image (2891).png>)

### Вкладка Цены

Чтобы внести цены на люверсы, щелкните мышкой на выбранные _свойства._

По умолчанию, для варианта когда свойство не выбрано, предусмотрено заполнение цен _Без свойств_.

![](<../../.gitbook/assets/image (500).png>)

Цена устанавливается от ед. изм. → цена за единицу.

![](<../../.gitbook/assets/image (2213).png>)

Через кнопку "Добавить цену", вы можете добавить несколько цен, в зависимости от количества люверсов, а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](../../settings/oplata/multivalyutnost.md), вы можете настроить разную валюту для комплектующих люверс.

Чтобы удалить цену из списка, нажмите кнопку "Удалить" напротив выбранной цены.

## Редактирование Люверса

Чтобы отредактировать данные, зайдите в нужный Люверс, щёлкнув мышкой на _название_.

![](<../../.gitbook/assets/image (2445).png>)

Внесите необходимые изменения в соответствующих вкладках.

Для удобства комплектующие Люверс можно копировать. Нажмите кнопку "Копировать" напротив нужного люверса.

![](<../../.gitbook/assets/image (2200).png>)

и дубликат появится в общем списке люверсов

![](<../../.gitbook/assets/image (2960).png>)

## Отключение люверсов в калькуляции

В общем списке, напротив каждого из люверсов есть переключатели "Выкл/Вкл."

![](<../../.gitbook/assets/image (3544).png>)

При помощи их вы можете отключить люверс в продукте на сайте, например, если комплектующие временно закончились.

В этом случае, отключенный люверс не отображается клиенту.

## Удаление люверсов

Для удаления люверса, нажмите напротив кнопку "Удалить"

![](<../../.gitbook/assets/image (2857).png>)

В случае, если люверс используется в каких-либо продуктах, система предупредит вас об этом.

![](<../../.gitbook/assets/image (2189).png>)

В предупреждении для удобства выводится список продукции, в калькуляции которой используется данный люверс.

Щелкнув на _название_, вы попадете сразу на продукт, где сможете удалить его.
