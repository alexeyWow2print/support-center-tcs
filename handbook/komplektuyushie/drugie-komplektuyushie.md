# Другие комплектующие

## Добавление Другого комплектующего&#x20;

Все комплектующие, не входящие в перечень разделов, можно внести отдельно в Другие комплектующие. Для этого в Комплектующие щелкните на нижний раздел Другие комплектующие.

![](<../../.gitbook/assets/image (2439).png>)



и нажмите "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (1457).png>)



### Вкладка Описание

В открывшемся окне заполните поле _Название\*_ и выберите параметры:

* _Группу_ (при необходимости)
* Плавное распределение цены (при необходимости)
* _Вес\*_ (в гр.)
* _Ед. измерения_ комплектующего: лист, м2, см2, шт., компл., г/м2,  пог.м, пог. см., пог. мм.
* Загрузите _картинку_ (минимальный размер 300x300px, jpg, gif, png, webp) и _иконку_ (минимальный размер 79x79px, jpg, gif, png, webp)
* _Описание_ (заполните поле)

Загруженные картинки и текст в поле _Описание_ будут отображаться на сайте, при наведении курсора мыши на параметр. Иконка ускорит поиск операции в папках или общем списке.

После внесения всех данных и загрузки изображений, нажмите "Сохранить".

После сохранения вкладки Описание, появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<../../.gitbook/assets/image (1209).png>)



### Вкладка Фотогалерея

#### Добавление изображений <a href="#dobavlenie-izobrazhenii" id="dobavlenie-izobrazhenii"></a>

Чтобы добавить изображения в Фотогалерею нажмите кнопку "Добавить" → "Начать загрузку" → "Загрузить".

Требования к загружаемым файлам: минимальный размер 643x300, картинки jpg, gif, png, webp

![](<../../.gitbook/assets/image (956).png>)

### Вкладка Свойства

{% hint style="info" %}
**Добавление свойства не является обязательным!** Если свойства не предусмотрены,  пропустите вкладку и переходите сразу к ценам.
{% endhint %}



Чтобы добавить свойства (цвет, материал и т.д..) нажмите кнопку "Добавить"

![](<../../.gitbook/assets/image (3111).png>)

В открывшемся окне будут отображены свойства из Справочник → Свойства → Свойства комплектующих (в данный раздел справочника вы можете самостоятельно добавить другие необходимые вам свойства: цвет, материал и т.д..).

![](<../../.gitbook/assets/image (3329).png>)

Выбранные свойства вы отображаются на сайте, если в Калькуляции продукта выбрана опция "Отображать свойства"**.**

Расположение  в калькуляции

![](<../../.gitbook/assets/image (1343).png>)



### Вкладка Цены

Чтобы внести цены на Другие комплектующие, щелкните мышкой на выбранные _свойства_.

![](<../../.gitbook/assets/image (2519).png>)

По умолчанию, для варианта когда свойство не выбрано, предусмотрено заполнение цен _Без свойств_.

Цена устанавливается от ед. изм. → цена за единицу.&#x20;

![](<../../.gitbook/assets/image (1965).png>)

Через кнопку "Добавить цену", вы  можете добавить несколько цен, в зависимости от количества комплектующего,  а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](../../settings/oplata/multivalyutnost.md), вы можете настроить разную валюту для созданных Других комплектующих.

Чтобы удалить цену из списка, нажмите кнопку "Удалить" напротив выбранной цены.

![](<../../.gitbook/assets/image (3737).png>)



## Редактирование Других комплектующих

Чтобы отредактировать данные, зайдите в нужный комплектующий, щёлкнув мышкой на _название._

![](<../../.gitbook/assets/image (2110).png>)

Внесите необходимые изменения в соответствующих вкладках.

Для удобства Другие комплектующие можно копировать. Нажмите кнопку "Копировать" напротив нужного комплектующего

![](<../../.gitbook/assets/image (476).png>)

и дубликат появится в общем списке:

![](<../../.gitbook/assets/image (2373).png>)



## Отключение Других комплектующих в калькуляции&#x20;

В общем списке, напротив каждого наименования есть переключатели "Выкл/Вкл."

![](<../../.gitbook/assets/image (2113).png>)

При помощи их вы можете отключить Другие комплектующие в продукте на сайте, например, если комплектующие временно закончились.&#x20;

В этом случае, отключенный вариант не отображается клиенту.



## Удаление Других комплектующих

Для удаления Других комплектующих, нажмите напротив нужного наименования кнопку "Удалить"

![](<../../.gitbook/assets/image (2141).png>)

В случае, если  комплектующий используется в каких-либо продуктах, система предупредит вас об этом.

![](<../../.gitbook/assets/image (2106).png>)

В предупреждении для удобства выводится список продукции, в калькуляции которой используется данный комплектующий.&#x20;

Щелкнув на _название_, вы попадете сразу на продукт, где сможете удалить его.
