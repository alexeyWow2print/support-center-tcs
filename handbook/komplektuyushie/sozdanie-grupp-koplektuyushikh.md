# Создание групп Комплектующих

Системой предусмотрена возможность группировать Комплектующие по следующему алгоритму (на примере комплектующего Пружина).

###

### **Создание Групп комплектующих**

В разделе для удобства предусмотрена группировка по созданным комплектующим.&#x20;

Создать группу можно через соответствующую кнопку "Добавить группу".&#x20;

![](<../../.gitbook/assets/image (2622).png>)



Заполните поле _Название_ и сохраните папку.

![](<../../.gitbook/assets/image (2431).png>)

После сохранения папка появился в общем списке.

![](<../../.gitbook/assets/image (1984).png>)



### **Группировка комплектующих**

Чтобы сгруппировать внесенные данные, в соответствующем комплектующем в поле _Группа_ выберите нужную, и нажмите "Сохранить"

![](<../../.gitbook/assets/image (1954).png>)

Комплектующие появятся в папке

![](<../../.gitbook/assets/image (2476).png>)

###

### **Редактирование и удаление папок**

Чтобы редактировать _название_ Группы (папки) комплектующих или удалить, нажмите соответствующие кнопки

![](<../../.gitbook/assets/image (3070).png>)



{% hint style="info" %}
**В случае удаления Группы, входящие в нее комплектующие не удаляются!** Комплектующие  попадают в общий список комплектующих.
{% endhint %}
