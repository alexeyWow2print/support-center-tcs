# Фольга

## Добавление комплектующего Фольга

Чтобы добавить фольгу, в Комплектующие щелкните на раздел Фольга

![](<../../.gitbook/assets/image (1460).png>)



и нажмите "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (3483).png>)



### Вкладка Описание

В открывшемся окне заполните поле *Название\** и выберите параметры:

-  *Группу* (при необходимости)

-  Плавное распределение цены (при необходимости)

-  *Вес\** (в гр.)

-  *Ед. измерения* комплектующего: лист, м2, см2, шт., компл., г/м2,  пог.м, пог. см., пог. мм.

-  Загрузите *картинку* (минимальный размер 300x300px, jpg, gif, png, webp) и *иконку* (минимальный размер 79x79px, jpg, gif, png, webp)

-  *Описание* (заполните поле)

Загруженные картинки и текст в поле *Описание* будут отображаться на сайте, при наведении курсора мыши на параметр. Иконка ускорит поиск операции в папках или общем списке.

После внесения всех данных и загрузки изображений, нажмите "Сохранить".

После сохранения вкладки Описание, появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<../../.gitbook/assets/image (1886).png>)



### Вкладка Фотогалерея

#### Добавление изображений 

Чтобы добавить изображения в Фотогалерею нажмите кнопку "Добавить" -> "Начать загрузку" -> "Загрузить".

Требования к загружаемым файлам: минимальный размер 643x300, картинки jpg, gif, png, webp

![](<../../.gitbook/assets/image (1394).png>)

####  

#### **Применение Фотогалереи** 

 Картинки загруженные во вкладку Фотогалереи отображаются в калькуляции продукта при выборе в Комплектующие-> Фольга -> Галерея -> Вкл. -> Применить.

![](<../../.gitbook/assets/image (1450).png>)

####  

#### **Удаление изображений**  

Чтобы удалить изображение нажмите кнопку "Удалить" ( ![](./folga.png){width=131px height=40px} ) напротив загруженного изображения. ​



### Вкладка Свойства

:::info 

**Добавление свойства не является обязательным!** Если свойства не предусмотрены,  пропустите вкладку и переходите сразу к ценам.

:::





Чтобы добавить свойства (цвет, материал и т.д..) нажмите кнопку "Добавить"

![](<../../.gitbook/assets/image (1850).png>)



В открывшемся окне будут отображены свойства из Справочник -> Свойства -> Свойства комплектующих (в данный раздел справочника вы можете добавить другие необходимые вам свойства: цвет, материал и т.д..).

![](<../../.gitbook/assets/image (3114).png>)



Выбранные свойства вы можете отображать на сайте, для этого в Калькуляции продукта выберите "Отображать свойства"**.**

Пример в калькуляции:

![](<../../.gitbook/assets/image (1841).png>)



### Вкладка Цены

Чтобы внести цены на фольгу, щелкните мышкой на выбранные *свойства*.

![](<../../.gitbook/assets/image (1882).png>)



По умолчанию, для варианта когда свойство не выбрано, предусмотрено заполнение цен *Без свойств*.

Цена устанавливается от ед. изм. -> цена за единицу. 

![](<../../.gitbook/assets/image (1874).png>)



Через кнопку "Добавить цену", вы можете добавить несколько цен, в зависимости от количества комплектующего, а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](./../../settings/oplata/multivalyutnost), вы можете настроить разную валюту для комплектующего Фольга.

Чтобы удалить цену из списка, нажмите кнопку "Удалить" напротив выбранной цены.

![](<../../.gitbook/assets/image (1843).png>)



## Редактирование комплектующего Фольга

Чтобы отредактировать данные, зайдите в нужную Фольгу, щёлкнув мышкой на *название*.

![](<../../.gitbook/assets/image (1817).png>)



Внесите необходимые изменения в соответствующих вкладках.

Для удобства комплектующие Фольга можно копировать. Нажмите кнопку "Копировать" напротив нужного комплектующего

![](<../../.gitbook/assets/image (1431).png>)



и дубликат появится в общем списке:

![](<../../.gitbook/assets/image (1825).png>)



## Отключение комплектующего Фольга в калькуляции 

В общем списке, напротив каждого наименования есть переключатели "Выкл/Вкл."

![](<../../.gitbook/assets/image (3537).png>)



При помощи их вы можете отключить фольгу в продукте на сайте, например, если комплектующие временно закончились. 

В этом случае, отключенный вариант не отображается клиенту.



## Удаление комплектующего Фольга

Для удаления фольги, нажмите напротив кнопку "Удалить".

![](<../../.gitbook/assets/image (782).png>)



В случае, если комплектующий используется в каких-либо продуктах, система предупредит вас об этом.

![](<../../.gitbook/assets/image (3066).png>)



В предупреждении для удобства выводится список продукции, в калькуляции которой используется данная фольга. 

Щелкнув на *название*, вы попадете сразу на продукт, где сможете удалить его.
