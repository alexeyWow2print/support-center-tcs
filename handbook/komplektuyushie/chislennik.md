# Численник

## Добавление комплектующего Численник

Чтобы добавить численник, в Комплектующие щелкните на раздел Численник

![](<../../.gitbook/assets/image (1887).png>)

и нажмите "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (3025).png>)

### Вкладка Описание

В открывшемся окне заполните поле _Название\*_ и выберите параметры:

* _Группу_ (при необходимости)
* Плавное распределение цены (при необходимости)
* _Вес\*_ (в гр.)
* _Ед. измерения_ комплектующего: лист, м2, см2, шт., компл., г/м2, пог.м
* Загрузите _картинку_ (минимальный размер 300x300px, jpg, gif, png, webp) и _иконку_ (минимальный размер 79x79px, jpg, gif, png, webp)
* _Описание_ (заполните поле)

Загруженные картинки и текст в поле _Описание_ будут отображаться на сайте, при наведении курсора мыши на параметр. Иконка ускорит поиск операции в папках или общем списке.

После внесения всех данных и загрузки изображений, нажмите "Сохранить".

После сохранения вкладки Описание, появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<../../.gitbook/assets/image (651).png>)

### Вкладка Свойства

{% hint style="info" %}
**Добавление свойства не является обязательным!** Если свойства не предусмотрены, пропустите вкладку и переходите сразу к ценам.
{% endhint %}

Чтобы добавить свойства (цвет, материал и т.д..) нажмите кнопку "Добавить"

![](<../../.gitbook/assets/image (741).png>)

В открывшемся окне будут отображены свойства из Справочник → Свойства → Свойства комплектующих (вы можете добавить туда другие необходимые вам свойства: цвет, материал и т.д..).

![](<../../.gitbook/assets/image (3081).png>)

Выбранные свойства вы можете отображать на сайте, для этого в Калькуляции продукта выберите "Отображать свойства".

Пример в калькуляции:

![](<../../.gitbook/assets/image (1839).png>)

### Вкладка Цены

Чтобы внести цены на численник, щелкните мышкой на выбранные _свойства_.

![](<../../.gitbook/assets/image (3499).png>)

По умолчанию, для варианта когда свойство не выбрано, предусмотрено заполнение цен _Без свойств_.

Цена устанавливается от ед. изм. → цена за единицу.

![](<../../.gitbook/assets/image (1285).png>)

Через кнопку "Добавить цену", вы можете добавить несколько цен, в зависимости от количества комплектующего, а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](../../settings/oplata/multivalyutnost.md), вы можете настроить разную валюту для комплектующего Численник.

Чтобы удалить цену из списка, нажмите кнопку "Удалить" напротив выбранной цены.

## Редактирование Численника

Чтобы отредактировать данные, зайдите в нужный Численник, щёлкнув мышкой на _название._

![](<../../.gitbook/assets/image (1876).png>)

Внесите необходимые изменения в соответствующих вкладках.

Для удобства комплектующие Численник можно копировать. Нажмите кнопку "Копировать" напротив нужного комплектующего

![](<../../.gitbook/assets/image (3461).png>)

и дубликат появится в общем списке:

![](<../../.gitbook/assets/image (1508).png>)

## Отключение Численника в калькуляции

В общем списке, напротив каждого численника есть переключатели "Выкл/Вкл."

![](<../../.gitbook/assets/image (1860).png>)

При помощи их вы можете отключить численник в продукте на сайте, например, если комплектующие временно закончились.

В этом случае, отключенный вариант не отображается клиенту.

## Удаление Численника

Для удаления численника, нажмите напротив кнопку "Удалить".

![](<../../.gitbook/assets/image (583).png>)

В случае, если комплектующие используется в каких-либо продуктах, система предупредит вас об этом.

![](<../../.gitbook/assets/image (1826).png>)

В предупреждении для удобства выводится список продукции, в калькуляции которой используется данный численник.

Щелкнув на _название_, вы попадете сразу на продукт, где сможете удалить его.
