# Лосьон/ароматизатор

## Добавление комплектующего Лосьон/ароматизатор

Чтобы добавить лосьон/ароматизатор, в Комплектующие щелкните на раздел Лосьон/ароматизатор

![](<../../.gitbook/assets/image (1473).png>)



и нажмите "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (3602).png>)



### Вкладка Описание

В открывшемся окне заполните поле _Название\*_ и выберите параметры:

* _Группу_ (при необходимости)
* Плавное распределение цены (при необходимости)
* _Вес\*_ (в гр.)
* _Ед. измерения_ комплектующего: лист, м2, см2, шт., компл., г/м2,  пог.м, пог. см., пог. мм. &#x20;
* Загрузите _картинку_ (минимальный размер 300x300px, jpg, gif, png, webp) и _иконку_ (минимальный размер 79x79px, jpg, gif, png, webp)
* _Описание_ (заполните поле)

Загруженные картинки и текст в поле _Описание_ будут отображаться на сайте, при наведении курсора мыши на параметр. Иконка ускорит поиск операции в папках или общем списке.

После внесения всех данных и загрузки изображений, нажмите "Сохранить".

После сохранения вкладки Описание, появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<../../.gitbook/assets/image (1891).png>)

### Вкладка Свойства

{% hint style="info" %}
**Добавление свойства не является обязательным!** Если свойства не предусмотрены,  пропустите вкладку и переходите сразу к ценам.
{% endhint %}

Чтобы добавить свойства (цвет, материал и т.д..) нажмите кнопку "Добавить"

![](<../../.gitbook/assets/image (1823).png>)



В открывшемся окне будут отображены свойства из Справочник → Свойства → Свойства комплектующих (вы можете добавить другие необходимые вам свойства: цвет, материал и т.д..).&#x20;

Выбранные свойства вы можете отображать на сайте, для этого в Калькуляции продукта выберите "Отображать свойства"**.**

Пример в калькуляции:

![](<../../.gitbook/assets/image (1054).png>)



### Вкладка Цены

Чтобы внести цены на лосьон/ароматизатор, щелкните мышкой на выбранные _свойства_.

![](<../../.gitbook/assets/image (1161).png>)

По умолчанию, для варианта когда свойство не выбрано, предусмотрено заполнение цен _Без свойств_.

Цена устанавливается от ед. изм. → цена за единицу.&#x20;

![](<../../.gitbook/assets/image (3472).png>)

Через кнопку "Добавить цену", вы  можете добавить несколько цен, в зависимости от количества комплектующего, а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](../../settings/oplata/multivalyutnost.md), вы можете настроить разную валюту для комплектующего Лосьон/ароматизатор.

Чтобы удалить цену из списка, нажмите кнопку "Удалить" напротив выбранной цены.



## Редактирование Лосьона/ароматизатора

Чтобы отредактировать данные, зайдите в нужный Лосьон/ароматизатор, щёлкнув мышкой на _название_.

![](<../../.gitbook/assets/image (3424).png>)

Внесите необходимые изменения в соответствующих вкладках.

Для удобства комплектующие Лосьон/ароматизатор можно копировать. Нажмите кнопку "Копировать" напротив нужного комплектующего

![](<../../.gitbook/assets/image (3484).png>)

и дубликат появится в общем списке:

![](<../../.gitbook/assets/image (2446).png>)



## Отключение Лосьона/ароматизатора в калькуляции&#x20;

В общем списке, напротив каждого лосьона/ароматизатора есть переключатели "Выкл/Вкл."

![](<../../.gitbook/assets/image (3330).png>)

При помощи их вы можете отключить лосьон/ароматизатор в продукте на сайте, например, если комплектующие временно закончились.

&#x20;В этом случае, отключенный вариант не отображается клиенту.



## Удаление Лосьона/ароматизатора

Для удаления лосьона/ароматизатора, нажмите напротив кнопку "Удалить"

![](<../../.gitbook/assets/image (1520).png>)

В случае, если комплектующие используется в каких-либо продуктах, система предупредит вас об этом.

![](<../../.gitbook/assets/image (3451).png>)

В предупреждении для удобства выводится список продукции, в калькуляции которой используется данный лосьон/ароматизатор.&#x20;

Щелкнув на _название_, вы попадете сразу на продукт, где сможете удалить его.
