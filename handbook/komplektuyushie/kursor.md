# Курсор

## Добавление комплектующего Курсор

Чтобы добавить курсор, в Комплектующие щелкните на раздел Курсор

![](<../../.gitbook/assets/image (1489).png>)



и нажмите "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (3252).png>)



### Вкладка Описание

В открывшемся окне заполните поле _Название\*_ и выберите параметры:

* _Группу_ (при необходимости)
* Плавное распределение цены (при необходимости)
* _Вес\*_ (в гр.)
* _Ед. измерения_ комплектующего: лист, м2, см2, шт., компл., г/м2,  пог.м, пог. см, пог. мм.
* Загрузите _картинку_ (минимальный размер 300x300px, jpg, gif, png, webp) и _иконку_ (минимальный размер 79x79px, jpg, gif, png, webp)
* _Описание_ (заполните поле)

Загруженные картинки и текст в поле _Описание_ будут отображаться на сайте, при наведении курсора мыши на параметр. Иконка ускорит поиск операции в папках или общем списке.

После внесения всех данных и загрузки изображений, нажмите "Сохранить".

После сохранения вкладки Описание, появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<../../.gitbook/assets/image (1836).png>)

### Вкладка Свойства

{% hint style="info" %}
**Добавление свойства не является обязательным!** Если свойства не предусмотрены,  пропустите вкладку и переходите сразу к ценам.
{% endhint %}



Чтобы добавить свойства (цвет, материал и т.д..) нажмите кнопку "Добавить"

![](<../../.gitbook/assets/image (2976).png>)

В открывшемся окне будут отображены свойства из Справочник → Свойства → Свойства комплектующих (вы можете добавить другие необходимые вам свойства: цвет, материал и т.д..).&#x20;

![](<../../.gitbook/assets/image (1461).png>)

Выбранные свойства вы можете отображать на сайте, для этого в Калькуляции продукта выберите "Отображать свойства"**.**

Пример в калькуляции

![](<../../.gitbook/assets/image (585).png>)



### Вкладка Цены

Чтобы внести цены на курсор, щелкните мышкой на выбранные _свойства._

![](<../../.gitbook/assets/image (3574).png>)

По умолчанию, для варианта когда свойство не выбрано, предусмотрено заполнение цен _Без свойств_.

Цена устанавливается от ед. изм. → цена за единицу.&#x20;

![](<../../.gitbook/assets/image (1852).png>)



Через кнопку "Добавить цену", вы можете добавить несколько цен, в зависимости от количества комплектующего, а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](../../settings/oplata/multivalyutnost.md), вы можете настроить разную валюту для комплектующего Курсор.

Чтобы удалить цену из списка, нажмите кнопку "Удалить" напротив выбранной цены.

![](<../../.gitbook/assets/image (3051).png>)

## Редактирование Курсора

Чтобы отредактировать данные, зайдите в нужный Курсор, щёлкнув мышкой на _название_.

![](<../../.gitbook/assets/image (1739).png>)

Внесите необходимые изменения в соответствующих вкладках.

Для удобства комплектующие Курсор можно копировать. Нажмите кнопку "Копировать" напротив нужного комплектующего

![](<../../.gitbook/assets/image (696).png>)

и дубликат появится в общем списке:

![](<../../.gitbook/assets/image (1758).png>)



## Отключение Курсора в калькуляции&#x20;

В общем списке, напротив каждого курсора есть переключатели "Выкл/Вкл."

![](<../../.gitbook/assets/image (1779).png>)

При помощи их вы можете отключить курсор в продукте на сайте, например, если комплектующие временно закончились.

В этом случае, отключенный вариант не отображается клиенту.



## Удаление Курсора

Для удаления курсора, нажмите напротив кнопку "Удалить"

![](<../../.gitbook/assets/image (1515).png>)

В случае, если  комплектующие используется в каких-либо продуктах, система предупредит вас об этом.

![](<../../.gitbook/assets/image (1834).png>)

В предупреждении для удобства выводится список продукции, в калькуляции которой используется данный курсор.&#x20;

Щелкнув на _название,_ вы попадете сразу на продукт, где сможете удалить его.
