## Добавление Товара (на примере товара Кружка)

Чтобы добавить Товар, в разделе Товары щелкните на нужный подраздел. В случае, если создаваемый товар не входит в указанные разделы, выберите для него Другие товары.



![](<./image (425).png>)



Чтобы создать новый товар нажмите "Добавить" в правом верхнем углу

![](<./image (2399).png>)



### Вкладка Описание

В открывшемся окне заполните обязательное поле *Название\** и выберите параметры:

-  *Группу* (при необходимости)

-  Плавное распределение цены (при необходимости)

-  *Вес\** (заполните в гр.)

:::info 

Единица измерения Товара всегда штуки (шт.)

:::



После сохранения вкладки Описание,  появится расширенная форма с дополнительными вкладками Фотогалерея/Свойства/Цены.

![](<./image (458).png>)





### Вкладка Фотогалерея

#### Добавление изображений

Чтобы добавить изображения в Фотогалерею нажмите кнопку "Добавить" -> "Начать загрузку" -> "Загрузить".

Требования к загружаемым файлам: минимальный размер 643x300, картинки jpg, gif, png, webp

![](<./image (1987).png>)

#### 

#### **Применение Фотогалереи**

Картинки загруженные во вкладку Фотогалереи отображаются в калькуляции продукта при выборе в Товар (Кружка) -> Галерея -> Вкл. -> Применить.

![](<./image (2125).png>)

#### 

#### **Удаление изображений**

Чтобы удалить изображение нажмите кнопку "Удалить" ( ![](<./image (3080).png>)) напротив загруженного изображения. ​





### Вкладка Свойства

:::info 

**Для Товара добавление свойства является ОБЯЗАТЕЛЬНЫМ! Без свойства невозможно добавление цены по вкладке Цены.**

:::





Чтобы добавить свойства (цвета) нажмите кнопку "Добавить".

![](<./image (2427).png>)



В открывшемся окне будут отображены свойства из Справочник -> Свойства -> Цвет (в данный раздел справочника вы можете добавить другие необходимые вам цвета).

![](<./image (485).png>)



### 

### Вкладка Цены

:::info 

Перед заполнением цены необходимо добавить Свойства Товара (цвет). В случае, если это не было сделано во вкладке Свойства, вам будет предложено заполнить выбрать цвет во вкладке Цены.

:::



Чтобы внести цены на Товар, щелкните мышкой на выбранные *свойства*.

Цена устанавливается от ед. изм. -> цена за единицу (т.е.  от шт.  цена за шт.).

![](<./image (2568).png>)



Через кнопку "Добавить цену", вы  можете добавить несколько цен, в зависимости от количества товара, а также предусмотреть скидки для групп клиентов.

В случае, если у вас установлен модуль ["Мультивалютность"](./../../settings/oplata/multivalyutnost), вы можете настроить для Товара разную валюту.

Чтобы удалить цену товара из списка, нажмите кнопку "Удалить" напротив выбранной цены.

![](<./image (1980).png>)



Во вкладке Цены вы можете добавить Свойства, нажав на кнопку "Добавить" под списком свойств.

![](<./image (3014).png>)



Выберите нужный цвет из выпадающего списка.

![](<./image (2378).png>)

После сохранения цвет отобразится и во вкладке Свойства, и во вкладке Цены, где вы сможете продолжить заполнение цен.



## Редактирование Товара

Чтобы отредактировать данные, зайдите в нужный Товар, щёлкнув мышкой на название.

![](<./image (2352).png>)



Внесите необходимые изменения в соответствующих вкладках.

Для удобства Товар можно копировать. Нажмите кнопку "Копировать" напротив нужного товара

![](<./image (2548).png>)



и дубликат появится в общем списке:

![](<./image (1205).png>)



## Удаление Товара

Для удаления Товара, нажмите напротив кнопку "Удалить".

![](<./image (2135).png>)



В случае, если Товар используется в каких-либо продуктах, система предупредит вас об этом.

![](<./image (1928).png>)



В предупреждении для удобства выводится список продукции, в калькуляции которой используется данный товар.

Щелкнув на название, вы попадете сразу на продукт, где сможете удалить его.
