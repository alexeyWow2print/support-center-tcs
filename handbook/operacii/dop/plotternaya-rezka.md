# Плоттерная резка

## **Добавление операции плоттерная резка**

Чтобы добавить новую операцию плоттерной резки, нажмите кнопку "Добавить" в правом верхнем углу интерфейса.

&#x20;Если нужно создать отдельную группу для операций резки, сначала нажмите "Добавить группу".

<figure><img src="../../.gitbook/assets/image (3898).png" alt=""><figcaption></figcaption></figure>

### **Вкладка Описание**

После создания операции откроется вкладка для заполнения параметров:

**Обязательные поля:**

* **Название:** укажите понятное название, отражающее суть операции.
* **Расчёт стоимости за:** выберите единицу расчёта: "лист" или "погонный метр".

**Дополнительные настройки:**

* **Плавное распределение цен:** включите этот параметр, чтобы цена автоматически, плавно распределялась между минимальным и максимальным значениями. Подробнее в статье о [плавном распределении цен](../materialy/bumaga/plavnoe-raspredelenie-ceny.md).
* **Группа:** выберите группу (папку) для хранения операции.
* **Валюта/Валюта себестоимости:** денежная единица, в которой производится расчёт. Отображается при условии, что у вас включен модуль "[Мультивалютность](../../settings/oplata/multivalyutnost.md)". По умолчанию выбран Российский рубль
* **Пуск машины/Приладка:** укажите фиксированную стоимость подготовки оборудования.
* **Себестоимость:** если требуется, то заполните себестоимость для пуска машины или приладки.
* **Группы клиентов:** при необходимости можно указать сколько будет стоить пуск или приладка машины для определенной группы клиентов.
* **Описание:** добавьте поясняющий текст, который будет отображаться в карточке товара при наведении на выбор параметра резки.

После заполнения нажмите "Сохранить".

### **Вкладка Цены**

#### **За лист**

При выборе варианта "за лист" система позволяет установить гибкие цены в зависимости от количества изделий на листе. Этот подход учитывает, что при увеличении количества элементов растет и число необходимых резов, что влияет на итоговую стоимость.

**Расчёт стоимости** изделий на листе выполняется в пределах указанных ценовых границ. Если количество изделий превышает установленный порог, цена автоматически пересчитывается, начиная со следующего ценового диапазона.

**Например:**\
Если заданы цены для 5 и 10 изделий на листе, то:

* от 1 до 5 изделий — применяется первая цена,
* от 6 и более — используется следующая цена (для 10 изделий).

**Как настроить:**

1. Нажмите кнопку "Добавить" в разделе цен
2. В открывшемся окне укажите количество изделий, размещаемых на одном листе
3. Подтвердите выбор кнопкой "Создать"

<figure><img src="../../.gitbook/assets/image (3902).png" alt=""><figcaption></figcaption></figure>

Для каждого добавленного варианта из раскрывающегося списка заполните:

* **ОТ (лист):** укажите минимальное количество листов, начиная с которого будет применяться данная цена. Первое значение автоматически устанавливается как 0, а для всех последующих ценовых вариантов вы можете задать нужное значение вручную.
* **Цена за лист:** это стоимость обработки одного листа материала при выбранном количестве изделий.
* **Себестоимость:** предназначено для ввода производственной себестоимости обработки одного листа
* **Группы клиентов:** вы можете при необходимости установить индивидуальные цены для конкретных категорий заказчиков.
* **Валюта:** если у вас активирован модуль "[Мультивалютность](../../settings/oplata/multivalyutnost.md)", в соответствующем поле вы сможете выбрать валюту, в которой будет отображаться указанная цена.

Добавлять новые ценовые уровни можно кнопкой "**Добавить цену**"

{% hint style="info" %}
Формула расчёта: \
Цена = (Количество изделий на листе × Цена за лист) × Количество листов в тираже
{% endhint %}

#### **За погонный метр (пог.м)**

При выборе варианта расчёта за погонный метр во вкладке "Цены" отображаются аналогичные поля, что и при расчёте за лист, с одним важным отличием: параметр количества изделий на листе отсутствует, поскольку стоимость определяется исключительно по метражу.

Для добавления нового ценового диапазона нажмите кнопку "Добавить цену". Укажите значения метража и соответствующую стоимость. При необходимости задайте дополнительные параметры (себестоимость, группы клиентов и т.д.)

<figure><img src="../../.gitbook/assets/image (3903).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Формула расчёта:\
Общая стоимость = (Периметр изделия) × (Цена за 1 пог.м.)
{% endhint %}



### **Вкладка Перевод**

Данный раздел становится доступным при активированном модуле "Мультиязычность". Подробная инструкция по добавлению новых языков доступна в руководстве "[Настройки мультиязычности](../../settings/drugie-nastroiki/nastroiki-multiyazychnosti/)".

В этом разделе вы можете локализовать два ключевых элемента: название операции плоттерной резки и её описание.

**Процесс редактирования перевода:**

1. Кликните по иконке карандаша напротив соответствующего поля
2. В открывшемся редакторе введите перевод текста
3. Сохраните внесенные изменения

<figure><img src="../../.gitbook/assets/image (3904).png" alt=""><figcaption></figcaption></figure>



## **Редактирование операции Плоттерная резка**

**Редактирование:**\
Нажмите на название операции → внесите изменения → нажмите "Сохранить".

**Копирование:**\
Нажмите на иконку "Копировать"![](<../../.gitbook/assets/image (3908).png>)- создастся дубликат с пометкой \[1] в конце названия.

**Удаление:**\
Нажмите на иконку корзины ![](<../../.gitbook/assets/image (3910).png>). Если операция используется в продукции, то система предупредит и покажет список связанных позиций.
