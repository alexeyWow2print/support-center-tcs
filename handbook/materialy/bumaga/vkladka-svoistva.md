# Вкладка Свойства

**Добавление свойства материала**

Чтобы добавить свойства в вкладке Свойства нажмите кнопку "Добавить"

![](<../../../.gitbook/assets/image (524).png>)

В открывшемся окне заполните следующие параметры из списка:

![](<../../../.gitbook/assets/image (3649).png>)

* **РАЗМЕР** (отображаются все форматы/размеры заполненные в Справочнике)

**Для единицы измерения ЛИСТ** в поле _Размер_ отображается размер: ширина х высота листа.

Размеры вносятся в Справочник → раздел Свойства → Формат → кнопка "Добавить" → Сначала вносится Формат

![](<../../../.gitbook/assets/image (3258).png>)

Затем, щелкните в списке на _название_ созданного Формата

![](<../../../.gitbook/assets/image (726).png>)

и через кнопку "Добавить"

![](<../../../.gitbook/assets/image (1166).png>)

внесите нужные **ширину и высоту листа** в ед. измерения миллиметры (мм.)

![](<../../../.gitbook/assets/image (3553).png>)

По такому алгоритму в один Формат можно внести любое количество Размеров, не нужно создавать на каждый размер новый формат.

**Для единицы измерения КВ. МЕТР (М2) в поле размер отображается ширина рулона.**

Ширина рулона вносятся в Справочник → раздел Свойства → Ширина рулона→ кнопка "Добавить"

По аналогии с Форматом, вносится сначала **Тип ширины рулона**

![](<../../../.gitbook/assets/image (3642).png>)

А затем через щелкнув в общем списке на название, зайдите в созданный тип ширины рулона

![](<../../../.gitbook/assets/image (2648).png>)

И через кнопку "Добавить"

![](<../../../.gitbook/assets/image (3527).png>)

введите нужную ширину листа/рулона. Ширина вводится в ед. измерения миллиметры (мм.)

![](<../../../.gitbook/assets/image (3526).png>)

* **ПЛОТНОСТЬ**

Плотность вносится в Справочник → Свойства → Плотность → кнопка "Добавить" →Заполните поле Значение и ед. измерения.

После сохранения данных, внесенное значение появится в списке плотностей в Свойствах.

Обращаем внимание, что плотность является обязательным параметром для заполнения, т.к. если плотность не будет указана - вес материала будет 0.

![](<../../../.gitbook/assets/image (3271).png>)

\
В калькуляции продукта вы можете выбрать отобразить или скрыть выбор плотности на сайте для клиента, а также озаглавить/подписать выбор плотности на сайте.

![](<../../../.gitbook/assets/image (669).png>)

<br>

* **ТОЛЩИНА**

Толщина вносится в Справочник → Толщина → кнопка "Добавить"→ заполните поле _Значение_ и выберите _ед. измерения_ (мм. или мкм.)

После сохранения данных, внесенное значение появится в списке плотностей в Свойствах.

![](<../../../.gitbook/assets/image (3416).png>)

* **Вид печати** (Универсальная, Цифровая, Офсетная). В случае выбора конкретного типа печати (Цифровая/Офсетная), бумага не будет доступна для другого типа печати.
* **ЦВЕТ**

Цвет заполняется в Справочник → Свойства → Цвет → кнопка "Добавить" → заполните поле _Название_, выберите _Группу_ (при необходимости) и загрузите Картинку. После внесения всех данных нажмите "Сохранить".

![](<../../../.gitbook/assets/image (1359).png>)

\
В калькуляции продукта вы можете отобразить цвет на сайте для клиента или скрыть, а также выбрать заголовок для цвета материала и отображение (выпадающий список, радио баттон, иконки, радио баттон с иконками).

![](<../../../.gitbook/assets/image (2225).png>)

* **ТЭГ**

Тег заполняется в Справочник → Свойства → Теги → кнопка "Добавить" → заполните поле _Название_ и выберите _Тип_.

![](<../../../.gitbook/assets/image (3656).png>)

**Изменение свойства материала**

Чтобы изменить Свойства материла, в правом углу напротив каждого формата нажмите кнопку "Редактировать" (карандаш)

![](<../../../.gitbook/assets/image (3540).png>)

По строкам: Плотность, Толщина, Тэг, Вид печати, Цвет откроются поля для замены выбранных ранее вариантов.

После внесения изменений нажмите галочку, в конце строки, чтобы подтвердить окончательно их.

![](<../../../.gitbook/assets/image (784).png>)

**Удаление свойства материала**

Чтобы удалить свойства материла нажмите на кнопку "Удалить" напротив нужной строки характеристик.

![](<../../../.gitbook/assets/image (3587).png>)

В случае, если данный формат с указанными характеристиками уже используется в продукции система выдаст сообщение: "Ошибка изменения размера материала" и предложит варианты:

* заменить на другой вариант и список материалов на выбор
* удалить материал из продукции.

![](<../../../.gitbook/assets/image (1248).png>)

Выберите подходящий для вас вариант и нажмите кнопку "Заменить".
