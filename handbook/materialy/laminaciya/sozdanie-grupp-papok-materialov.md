# Создание Групп/папок материалов

**Создание Групп/папок материалов**

В разделе для удобства предусмотрена группировка по внесенным материалам ламинации. Создать группу можно через соответствующую кнопку «Добавить группу».

![](<../../../.gitbook/assets/image (1557).png>)

Заполните поле _Название_ и сохраните папку.

![](<../../../.gitbook/assets/image (1396).png>)

**Группировка материала**

Чтобы сгруппировать внесенные данные, в соответствующем материале ламинация в поле _Группа_ выберите нужную, и нажмите «Сохранить»

![](<../../../.gitbook/assets/image (1330).png>)

**Редактирование и удаление папок**

Чтобы редактировать _название_ Группы/папки или удалить, нажмите соответствующие кнопки

![](<../../../.gitbook/assets/image (3629).png>)

{% hint style="info" %}
**В случае удаления Группы, входящие в нее материалы не удаляются!** Материалы попадают в общий список материалов.
{% endhint %}
