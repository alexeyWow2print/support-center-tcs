# Создание групп доп. операций

Системой предусмотрена возможность группировать дополнительные операции по следующему алгоритму (на примере операции Резка).

###

### **Создание Групп доп. операций**

В  разделе для удобства предусмотрена группировка по созданным доп. операциям. Создать группу можно через соответствующую кнопку "Добавить группу".&#x20;

![](<../../.gitbook/assets/image (1027).png>)

Заполните поле _Название_ и сохраните папку.

После сохранения папка появился в общем списке.

![](<../../.gitbook/assets/image (2533).png>)



### **Группировка доп.операций**

Чтобы сгруппировать внесенные данные, в соответствующей доп.операции в  поле _Группа_ выберите  нужную, нажмите кнопку «Сохранить»

![](<../../.gitbook/assets/image (1749).png>)

и доп. операция переместится в указанную папку

![](<../../.gitbook/assets/image (3420).png>)

###

### **Редактирование и удаление папок**

Чтобы редактировать _название_  Группы (папки) или удалить, нажмите соответствующие кнопки

![](<../../.gitbook/assets/image (3505).png>)



{% hint style="info" %}
**В случае удаления Группы, входящие в нее доп. операции не удаляются!** Операции попадают в общий список до&#x43F;**.** операций.
{% endhint %}
