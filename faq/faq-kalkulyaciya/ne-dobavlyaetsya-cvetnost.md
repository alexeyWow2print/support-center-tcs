# Не добавляется цветность

## Не добавляется цветность 1+0/1+1

При попытке добавить в калькуляцию цветность печати 1+0 и 1+1 в списке нет ничего, кроме 4+0/4+4 и текста ошибки "Для конструкторского продукта доступны только 4+0 и 4+4":

![Количество сторон. Текст ошибки при добавлении.](<../../.gitbook/assets/image (1025).png>)

**Решение:**

{% hint style="info" %}
Чтобы была возможность добавлять в продукт любые типы цветности (1+0/1+1/4+1 и т.д.), он не должен содержать Конструктора макетов.
{% endhint %}

В некоторых продуктах может использоваться конструктор макетов.\
Конструктор макетов позволяет клиенту создать макет онлайн с нуля. При наличии Конструктора макетов, в продукте можно использовать цветность только 4+0 и 4+4.

Данный конструктор добавляется в продукт через модуль "Размеры" и выглядит следующим образом:

![Модуль "Размеры". Конструктор макетов](<../../.gitbook/assets/image (591).png>)

Следом необходимо заменить имеющиеся размеры с конструктором, на идентичные размеры без конструктора. Для этого необходимо перейти в "Добавить размер" и вместо размеров с конструктором добавить размеры без конструктора макетов.

* "Индивидуальный" — дает возможность вручную выставить желаемые размеры на вкладке "Размеры". В этом случае "Конструктор" не создается
* "Стандартный" — предоставляет на выбор готовые варианты размеров, вкладка "Размеры" будет заполнена автоматически, но важно выбрать вариант "Без конструктора"

После того как размеры продукта **без** конструктора будут добавлены — необходимо удалить старые размеры с конструктором.

В результате появится возможность добавлять те цветности печати, которые имеются в свойствах печатной машины

![Количество сторон. Все варианты цветности](<../../.gitbook/assets/image (3006).png>)
