# Теги (фильтр продукции)

## Общие положения

_Вы можете выделить продукт из общего списка с помощью тега, тег может передавать информацию о наличии скидки (Скидка), стоимости продукта по отношению к аналогичным (Выгодная цена), спрос на продукт (Хит продаж) и тому подобные._\
&#xNAN;_&#x422;акже с помощью тегов можно отобразить на странице категории фильтр, который позволит сортировать список продуктов по заданным условиям._

Варианты использования тегов:

### Создание тегов

Теги можно найти в разделе продукция над кнопкой «Добавить новый продукт». Кликнув на «Теги», откроется страница со всеми созданными тегами и группами тегов.

![](<../.gitbook/assets/image (3358).png>)

Чтобы создать тег, необходимо нажать на «Добавить тег», откроется следующее окно с настройками:

![](<../.gitbook/assets/image (935).png>)

* **Имя**\
  Имя тега.<br>
* **Группа**\
  Группа объединяющая в себе теги, является основным элементом при составлении [фильтра продукции](tegi-filtr-produkcii.md#filtr-produkcii).<br>
* **Избранный**\
  Именно данный параметр отвечает за отображение тега на сайте. При его активации, тег начнет отображаться на тизере продукта и на фотографии в калькуляторе типа «Карточка товара» <img src="../.gitbook/assets/image (1541).png" alt="" data-size="original"><br>
* **Выделить цветом**\
  Тег, имеющий параметр «Избранный», может быть выделен другим цветом. Цвет текста выбирается в разделе «[Элементы дизайна](../settings/dizain-saita/elementy-dizaina.md#cvetovoe-oformlenie)» <img src="../.gitbook/assets/image (3605).png" alt="" data-size="original">

### Добавление тегов в продукт

Сначала необходимо [создать теги](tegi-filtr-produkcii.md#sozdanie-tegov), отметить в них параметр «Избранный» и затем добавить их в соответствующий продукт. Для этого перейдем в настройки [продукта](produkty/#vkladka-obshie) на вкладку «Общие».

Теги добавляются в соответствующее поле «Теги», достаточно кликнуть на поле ввода и выбрать необходимый тег из списка или начать вводить название тега, после чего кликнуть на появившееся название в выпадающем списке.

![](<../.gitbook/assets/Добавление тегов 1.0.gif>)

### Фильтр продукции

Фильтры на сайте позволят клиенту сортировать представленную категорию продуктов по заданным условиям.

![](<../.gitbook/assets/Фильтр продукции 1.0.gif>)

В роли группы фильтров (Продукт, Тираж, Тип печати и Дополнительно) выступают группы тегов. Группы тегов в свою очередь можно найти в разделе «Продукция» → «[Теги](tegi-filtr-produkcii.md#sozdanie-tegov)».

Фильтры и группы фильтров располагаются в том порядке, в котором они расположены в админ-панели сайта.\
Сортировать между собой их можно путем перетаскивания курсором мыши.\
Имеется два варианта отображения фильтров:

Фильтр отображается только в том случае, если хоть один продукт категории обладает соответствующим тегом.
{% endhint %}

### Отображение фильтра в категории

{% hint style="info" %}
Фильтр в категории продуктов по умолчанию всегда отображается.\
Отображение того или иного фильтра для конкретной категории можно настроить вручную.
{% endhint %}

Настроить отображение фильтров можно в настройках категории.\
Для этого перейдите в редактирование конкретной категории — «Редактировать категорию», далее на вкладку «Фильтр»

![](<../.gitbook/assets/image (2531).png>)

При клике на «Настроить группы тегов вручную», список групп станет доступен для редактирования, можно отметить те группы, которые будут отображаться в редактируемой категории.
