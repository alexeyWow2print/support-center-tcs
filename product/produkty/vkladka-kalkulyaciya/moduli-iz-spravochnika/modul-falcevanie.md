# Модуль «Фальцевание»

### **Добавление модуля в калькуляцию**

Чтобы добавить операцию фальцевания в расчет стоимости продукта, выполните следующие шаги:

1. В правом нижнем углу экрана нажмите кнопку «Плюсик».
2. В появившемся меню выберите пункт «Добавить модуль».
3. В открывшемся окне укажите параметры:
   * **Деталь:** Выберите из списка деталь, к которой будет относиться фальцевание.
   * **Тип модуля:** Установите значение «Постпечать».
   * **Операция:** В списке найдите и выберите «Фальцевание».
4. Нажмите кнопку **«Сохранить»**.

После сохранения модуль появится в структуре калькуляции выбранной детали.

### **Вкладка «Операция Фальцевание»**

#### **Верхняя панель (**&#x43D;астройки отображения на сайт&#x435;**)**

*   **Показать на сайте:** Главный переключатель. Если он активен, блок с фальцеванием будет отображаться в карточке товара на сайте.

    При включении становятся доступны следующие поля:

    * **Заголовок на сайте:** Позволяет изменить стандартное название модуля (например, вместо «Фальцевание» указать «Сгибание»).
    * **Выбор параметра:**
      * _Одиночный:_ Позволяет выбрать в карточке товара только один вариант из предложенных.
      * _Множественный:_ Позволяет выбрать несколько вариантов. При этом в калькуляции пропадает вариант «Без параметра».
    * **Отображение:** Выбор внешнего вида опций на сайте (Выпадающий список, Радио баттон, Иконки, Радио баттон с иконками, Плитка).
    * **Описание:** Текст, который будет отображаться в карточке товара под заголовком модуля.

    **• Показывать заголовок на сайте:** Управляет отображением заголовка модуля на странице сайта.

<figure><img src="../../../../.gitbook/assets/image (4067).png" alt=""><figcaption></figcaption></figure>

* **Показывать группировки на сайте:** Если у вас настроены группировки операций, этот переключатель позволяет отобразить их для выбора клиенту.
  * При включении становятся доступны следующие поля:
    * **Заголовок для группировок:** Название блока с группировками.
    * **Отображение:** Формат показа группировок (Список, Радио баттон, Иконки и т.д.).
* **Галерея:** Включает или отключает показ блока с изображениями под модулем. Фотографии загружаются в настройках самой операции на вкладке **«Фотогалерея»**.

### **Настройка операций модуля** <a href="#nastroika-operacii-modulya" id="nastroika-operacii-modulya"></a>

В данном разделе происходит управление конкретными вариантами фальцевания.

По умолчанию в списке присутствует **«Вариант без параметра»**, который также можно активировать или отключить.

Для добавления новой операции нажмите кнопку **«Добавить»** и выберите из списка или создайте операцию «Фальцевание» (подробнее по [ссылке](../../../../handbook/dop.-operacii/falcevanie.md))

<figure><img src="../../../../.gitbook/assets/image (4060).png" alt=""><figcaption></figcaption></figure>

#### **Параметры добавленной операции:**

* **Иконка**: изображение, отображаемое на сайте при соответствующих формах отображения (иконки, радио баттон с иконками или плитка).
* **Название варианта параметра:** Наименование, которое увидит клиент на сайте.
* **Комментарий:** Внутренняя заметка для менеджеров (отображается только в панели управления).
* **Количество:** Коэффициент, на который умножается стоимость операции. По умолчанию равен 1.

<figure><img src="../../../../.gitbook/assets/image (4061).png" alt=""><figcaption></figcaption></figure>

Для каждого варианта в списке доступны следующие действия:

* **Переключатель (Вкл/Выкл):** Позволяет временно скрыть вариант из расчетов и с сайта без его удаления.
* **Описание (i):** Пояснение к данному типу фальцевания. Отображается как всплывающая подсказка при наведении на название на сайте.
* **Удалить (Корзина):** Полностью удаляет вариант из модуля.
* **Перемещение (≡):** Позволяет изменить порядок отображения вариантов в списке на сайте.
* **Настройка (Двойная скрепка):** Переход к детальной настройке выбранного варианта фальцевания (см. статью [«Фальцевание»](../../../../handbook/dop.-operacii/falcevanie.md)).

<figure><img src="../../../../.gitbook/assets/image (4062).png" alt=""><figcaption></figcaption></figure>

#### Управление модулем <a href="#upravlenie-modulem" id="upravlenie-modulem"></a>

В нижней части каждого модуля расположена панель управления:

* **Кнопка «Применить»:** Становится активной (зеленой) при внесении изменений в настройки. **Обязательно нажимайте эту кнопку для сохранения конфигурации модуля.**
* **Кнопка «Удалить» (Корзина):** Полностью удаляет модуль «Фальцевание» из калькуляции.
