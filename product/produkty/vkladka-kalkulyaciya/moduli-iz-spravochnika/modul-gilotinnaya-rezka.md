# Модуль «Гильотинная резка»

### **Добавление модуля «Гильотинная резка» в калькуляцию продукта**

Для добавления операции в калькуляцию продукта выполните следующие действия:

1. В интерфейсе редактирования продукта нажмите на значок «+» в правом нижнем углу экрана.
2. В открывшемся меню выберите **«Добавить модуль»**.
3. В диалоговом окне укажите деталь, к которой необходимо прикрепить операцию.
4. Выберите тип модуля: **«Постпечать»**.
5. В списке «Операция» укажите **«Гильотинная резка»**.
6. При необходимости к операции можно сразу прикрепить комплектующие, которые будут отображаться верхней вкладке «Комплектующие» данного модуля.
7. Нажмите **«Сохранить»**. Модуль будет добавлен в структуру калькуляции вашего продукта.

### **Настройка отображения модуля в карточке товара на сайте**

В верхней части добавленного модуля расположены ключевые переключатели для управления его отображением в карточке товара:

* **«Показывать на сайте»**: Активация данного параметра делает модуль видимым в карточке товара. При включении становятся доступными для редактирования следующие поля:
  * **«Заголовок на сайте»** – позволяет задать пользовательское название модуля.
  * **«Выбор параметров»** – позволяет разрешить заказчику выбрать несколько вариантов резки или только один.
  * **«Отображение»** – определяет формат отображения операции клиенту: выпадающий список, радио баттон, иконки, радио баттон с иконками или плитка.
  * **«Описание»** – поле для ввода поясняющего текста, который будет отображаться под заголовком.

{% hint style="info" %}
_Примечание_: Если переключатель выключен, то расчёт стоимости будет производиться автоматически или на основе первой активной операции из списка «Настройки доп. операций», без отображения выбора клиенту.
{% endhint %}

* **«Показывать заголовок на сайте»**: Управляет видимостью заголовка модуля непосредственно на странице сайта.

<figure><img src="../../../../.gitbook/assets/image (4045).png" alt=""><figcaption></figcaption></figure>

* **«Галерея»**: Включает или отключает показ галереи изображений в карточке товара. При активации под модулем резки отображаются фотографии, загруженные в настройках операции в вкладке «Фотогалерея».

### **Настройка операций модуля**

В данном разделе происходит управление конкретными вариантами услуг гильотинной резки.

* По умолчанию в списке присутствует **«Вариант без параметра»**, который также можно активировать или отключить.
* Для добавления новой операции нажмите кнопку **«Добавить»** и выберите из списка или создайте операцию «Гильотинная резка» ([подробнее о создании операций в отдельной статье](../../../../handbook/dop.-operacii/rezka.md)).

<figure><img src="../../../../.gitbook/assets/image (78).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>

* Для каждой операции доступен следующий набор настроек:
  * **Загрузка иконки** – изображение, отображаемое на сайте при соответствующих формах отображения (иконки, радио баттон с иконками или плитка).
  * **«Название варианта параметра»** – своё название операции для сайта (по умолчанию используется наименование из справочника).
  * **«Комментарий»** – внутренняя заметка, видимая только в интерфейсе админ-панели.
  * **«Группировка»** – привязка операции к определённой группе для фильтрации в карточке товара.
  * **«Количество»** – коэффициент, на который умножается итоговая стоимость, рассчитанная для данной операции.
  * **«Кол-во листов в стопке»** – параметр, доступный при настройке операции с расчётом стоимости за штуку. Позволяет выбрать автоматический расчёт или задать количество листов вручную.
  * **Переключатель** – включает или отключает отображение конкретной операции на сайте.
  * **«Описание»** – текст-подсказка, появляющаяся при наведении курсора на название операции в карточке товара на сайте.
  * **«Вспомогательная операция»** – позволяет привязать к основной дополнительную услугу (например, проклейка), которая будет рассчитываться только при выборе клиентом основной операции.
  * **«Удалить»** – удаляет операцию из списка модуля.
  * **Перемещение** – изменяет порядок операций в списке.
  * **Настройка** – открывает [детальные параметры](../../../../handbook/dop.-operacii/rezka.md#vkladka-opisanie) самой операции «Гильотинная резка».

### **Управление модулем**

В нижней части интерфейса модуля расположена панель управления:

* Кнопка **«Применить»** подсвечивается зелёным при внесении некоторых изменений в настройки модуля. Для сохранения конфигурации необходимо нажать эту кнопку.
* Кнопка **«Удалить»** (значок корзины) предназначена для полного удаления модуля «Гильотинная резка» со всеми операциями из калькуляции продукта.
