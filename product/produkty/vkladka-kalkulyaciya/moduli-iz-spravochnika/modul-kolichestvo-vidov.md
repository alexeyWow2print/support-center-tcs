# Модуль «Количество видов»

Модуль позволяет загрузить несколько макетов для одного тиража и гибко распределить общее количество изделий между этими макетами.

### **Настройка модуля**

Перед использованием модуль можно настроить с помощью двух переключателей и задания диапазона:

* **Показывать на сайте**: включает отображение поля для ввода количества видов и переключателя на сайте для клиента.
* **Показывать заголовок на сайте**: отображает название модуля в калькуляции на сайте.
* **Диапазон макетов**: ограничьте возможное количество загружаемых макетов, указав значения **ОТ** и **ДО**.

### **Как работать с модулем (окно загрузки макета)**

Нажмите кнопку «Загрузить макет». После успешной загрузки первого макета в правой части экрана появится панель для распределения тиража.

<figure><img src="../../../../.gitbook/assets/image (392).png" alt=""><figcaption></figcaption></figure>





### **Заполнение информации о макете**

Под каждым загруженным макетом находятся два обязательных поля:

* **Наименование вида**: Это название будет подставлено в скачиваемый макет, если в  [шаблоне названия макета](../../../../settings/drugie-nastroiki/rabota-s-maketom/nazvanie-maketov.md#sozdanie-shablona) выбрано «Наименование вида/макета».
* **Кол-во**: Укажите, какая часть тиража должна быть напечатана с этим макетом

<figure><img src="../../../../.gitbook/assets/image (364).png" alt=""><figcaption></figcaption></figure>





#### **Распределение тиража**

Справа отображается общий **Тираж** (указанный при оформлении заказа) и динамически изменяемый **Остаток**.&#x20;

Вносите значения в поля «Кол-во» для каждого макета, пока остаток не станет равен **0**. Только в этом случае можно будет продолжить оформление заказа.

<figure><img src="../../../../.gitbook/assets/image (365).png" alt=""><figcaption></figcaption></figure>





#### **Автоматизация (опционально)**

Для ускорения процесса заполнения используйте две опции:

* **«Использовать имя файла для наименования вида»**: автоматически присваивает макетам названия на основе имен загруженных файлов.
* **«Распределить тираж равномерно»**: система сама разделит тираж между всеми загруженными макетами. Если тираж не делится нацело, остаток распределится по макетам сверху вниз.

<figure><img src="../../../../.gitbook/assets/image (366).png" alt=""><figcaption></figcaption></figure>





### **Завершение загрузки**

После того как все поля заполнены и остаток тиража равен нулю, нажмите кнопку **«К согласованию макетов»**, чтобы продолжить оформление заказа.

### **Просмотр заказа в админ-панели**

После оформления заказ появится в админ-панели.

<figure><img src="../../../../.gitbook/assets/image (394).png" alt=""><figcaption></figcaption></figure>

* Для просмотра загруженных изображений используйте стрелки **«влево»** и **«вправо»**.
* Обложка (если она была добавлена) не отображается в превью, но её можно скачать, нажав на кнопку **«Макет»** рядом с ней.
* Чтобы скачать все макеты разом, нажмите **«Скачать макеты всех деталей»** или выберите в меню **«Архив макетов» → «Скачать»**.
