# Модуль «Комплектующие»

Модуль предназначен для добавления в калькуляцию готовых компонентов, которые не требуют сложного расчета (например, фурнитура или элементы отделки).



### Добавление модуля в калькуляцию

Чтобы добавить комплектующие в расчет стоимости продукта:

1. В правом нижнем углу экрана нажмите кнопку **«Плюсик»**.
2. В появившемся меню выберите пункт **«Добавить модуль»**.
3. В открывшемся окне укажите параметры:
   * **Деталь** — выберите из списка деталь, к которой будет относиться модуль.
   * **Тип модуля** — установите значение **«Комплектующие»** (выбрано по умолчанию).
   * **Комплектующие** — выберите готовую группу со специальными комплектующими или пункт _«Другие комплектующие»_ для самостоятельной настройки.
4. Нажмите кнопку **«Сохранить»**.

После сохранения модуль появится в структуре калькуляции выбранной детали.

### **Настройка отображения на сайте (Верхняя панель)**

В этом блоке настраивается внешний вид модуля в карточке товара:

* **Показывать на сайте** — Главный переключатель. Если активен, блок «Комплектующие» отображается в карточке товара.
* **Заголовок на сайте** — Позволяет изменить стандартное название модуля (например, с «Комплектующие» на «Вставки»).
* **Отображение** — Выбор формата опций на сайте: выпадающий список, радио-баттон, иконки, плитка и др.
* **Описание** — Текст-пояснение, который будет отображаться под заголовком модуля.
* **Галерея** — Включает показ галереи изображений (фотографии загружаются в настройках операции, вкладка **«Фотогалерея»**).
* **Показывать заголовок на сайте** — Управляет видимостью заголовка модуля.

#### **Отображать свойства**

При активации этой опции [свойства комплектующих](../../../../handbook/svoistva/untitled-1.md) будут видны в карточке заказа. Доступны настройки:

* **Заголовок свойства** — название блока.
* **Отображение свойства** — формат показа (выпадающий список, иконки, плитка и т.д.).

<figure><img src="../../../../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>





#### **Отображать поле ввода**

Если клиент должен самостоятельно указывать количество (например, площадь лакирования), активируйте этот переключатель. После этого появятся поля:

* **Заголовок для поля ввода** — название поля для клиента.
* **Минимальное** и **Максимальное значение** — диапазон допустимых значений.

<figure><img src="../../../../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>





#### **Показывать группировки на сайте**

Если у вас есть группировки операций, этот переключатель позволит отобразить их для выбора клиенту. При включении появляются два поля:

* **Заголовок для группировок** — название блока с группировками.
* **Отображение** — формат показа группировок (список, радио-кнопки, иконки и т.д.).

<figure><img src="../../../../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>





#### **Отображать текстовое поле**

Включает возможность изменить поле для ручного ввода текста клиентом (например, комментарий к материалу). При активации появляются:

* **Заголовок для текстового поля** — название поля, которое увидит клиент.
* **Подпись текстового поля** — отображается под заголовком.

<figure><img src="../../../../.gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>





### **Настройка комплектующих**

**Добавление операции:**

В этом блоке добавляются конкретные позиции (операции) комплектующих.

* По умолчанию добавлен **«Вариант без параметра»**.
* Чтобы добавить новое комплектующее, нажмите **«Добавить»** и выберите существующую операцию из списка или создайте новую (подробнее в статье [_«Комплектующие»_](../../../../handbook/komplektuyushie/)).

<figure><img src="../../../../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>





#### **Настройка добавленной операции**

* **Иконка** — Изображение, отображаемое на сайте при выборе формата «Иконки», «Плитка» или «Радио-баттон с иконками».
* **Название варианта параметра —** Наименование, которое увидит клиент на сайте.
* **Комментарий —** Внутренняя заметка для менеджеров (видна только в панели управления).
* **Отображать текстовое поле —** Включает отдельное текстовое поле для данного варианта.
* **Выбор кол-ва —** Настройка учета количества:\
  • **Фиксированный** — количество применяется к тиражу по умолчанию (без выбора клиентом). При выборе появляется строка для указания значения.\
  • **Вручную** — клиент самостоятельно указывает количество в заказе.
* **Свойства** — Если в настройках комплектующего добавлены [свойства](../../../../handbook/svoistva/untitled-1.md), то они отобразятся в этом поле, где их можно включить, отключить и сортировать

<figure><img src="../../../../.gitbook/assets/image (4122).png" alt=""><figcaption></figcaption></figure>





* **Переключатель** – включает или отключает отображение конкретной операции на сайте.
* **«Описание»** – текст-подсказка, появляющаяся при наведении курсора на название операции в карточке товара на сайте.
* **«Удалить»** – удаляет операцию из списка модуля.
* **Перемещение** – изменяет порядок операций в списке.
* **Настройка** – открывает  детальные параметры самой операции «Комплектующие».

<figure><img src="../../../../.gitbook/assets/image (4123).png" alt=""><figcaption></figcaption></figure>





### **Управление модулем**

В нижней части интерфейса модуля расположена панель управления:

* Кнопка **«Применить»** подсвечивается зелёным при внесении некоторых изменений в настройки модуля. Для сохранения конфигурации необходимо нажать эту кнопку.
* Кнопка **«Удалить»** (значок корзины) предназначена для полного удаления модуля «Комплектующие» со всеми операциями из калькуляции продукта.
