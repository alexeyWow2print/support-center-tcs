# Детали

### **Добавление детали**

Чтобы добавить модули из Справочника, предварительно нужно добавить Деталь. Продукт может состоять из любого количества деталей, необходимого вам для формирования калькуляции продукта.&#x20;

Чтобы добавить деталь, нажмите на знак "**Плюс"** в правом нижнем углу и выберите "Добавить деталь".&#x20;

<figure><img src="../../../../.gitbook/assets/image (396).png" alt=""><figcaption></figcaption></figure>

В открывшемся окне выберите деталь и основание для расчета калькуляции (в чем будет считаться деталь - лист, м2 или шт.)

![](<../../../../.gitbook/assets/image (640).png>)



Если деталь является имеет много страниц, например каталоги, блокноты, отметьте галочкой условие "_Многостраничная деталь"_ (подробнее можно посмотреть [_тут_](./#mnogostranichnost))_._ После завершения нажмите "Сохранить".

Вы можете сами добавлять детали, отображаемые в списке Деталь. Для этого в Справочнике → Продукция → Детали → Добавить введите название детали и при необходимости описание к ней.\
\
Что бы **скопировать всю деталь** со всеми содержащимися в них модулями, нажмите на значок Копировать ![](<../../../../.gitbook/assets/image (398).png>)

Чтобы **удалить деталь** из калькуляции продукта нажмите "Удалить" (Корзина).

<figure><img src="../../../../.gitbook/assets/image (397).png" alt=""><figcaption></figcaption></figure>



### **Размеры детали**

![](<../../../../.gitbook/assets/image (594).png>)

Каждая деталь содержит обязательный модуль "Размеры".&#x20;

Если размер детали равен размеру продукта, введенного в Основных модулях, то его можно не добавлять/заполнять. Система по умолчанию будет считать размер детали равным основному размеру.

В случае, если Размеры детали отличаются от основных, они  вводятся аналогично размерам в Основных модулях.&#x20;

Размеры деталей можно как и основные отобразить на сайте для выбора клиентом через переключатели "Показывать на сайте" и "Показывать заголовок на сайте".&#x20;

![](<../../../../.gitbook/assets/image (941).png>)



Если размер детали не нужно отображать на сайте для выбора, отображение размера можно закрыть. В этом случае, воспользуйтесь сопоставлением размеров продукции к размерам детали.

![](<../../../../.gitbook/assets/image (2546).png>)

Сопоставив размер продукта с конкретным размером детали, вы можете составить правило сопоставления для оставшихся общих размеров: уменьшить/увеличить  на (мм) или в (раз) и указать нужное значение. Правило будет приценятся к основному размеру.&#x20;
