# Тиражность

Тиражность - модуль, отвечающий за введение /отображение количества заказываемого товара.

Рассмотрим подробнее алгоритм заполнения полей модуля:

<figure><img src="../../../../.gitbook/assets/image (3843).png" alt=""><figcaption></figcaption></figure>

### Заголовок на сайте

При заполнении поля _Заголовок на сайте_, для клиента будет видно только выборе варианта ввода количества _Ручной ввод_.

Для подтверждения введенного вручную тиража в карточке товара, нажмите Enter или кликните на любое другое место на странице, тогда введённый тираж применится и рассчитается стоимость.

<figure><img src="../../../../.gitbook/assets/image (3844).png" alt=""><figcaption></figcaption></figure>

### Минимальная стоимость

Поле _Минимальная стоимость (в руб.)_ при заполнении данного поля, калькуляция не будет считаться ниже введенного значения.

Пример: на сайте продукт Визитки цена за шт. высчитывается исходя из цены печати, бумаги, резки. Но вы не хотите продавать продукт ниже чем за 1000 руб.\
Установив в калькуляции продукта минимальную стоимость 1000 руб. на сайте будет отображать цена 1000 руб. за любой тираж, пока по калькуляции не выйдет более 1000 руб.\
Визуально на сайте будет так:

### Показывать цену за 1 шт.

При выборе _Показывать цену за 1 шт._ цена за единицу продукции будет показываться на сайте

<figure><img src="../../../../.gitbook/assets/image (3846).png" alt=""><figcaption></figcaption></figure>

### Минимальная цена за тираж

При выборе _Минимальной цены за тираж_ и указании тиража, введенные данные будут отображаться в категории и в виджете "Рекомендуемые продукты". Информация обновляется каждые три часа.

### **Выбор вариантов**

Предусмотрено два варианта внесения количества продукции:

* **Выбор из списка**

При выборе из списка вы носите тиражи продукции, при этом предусмотрено введение фиксированных цен на весь тираж с учетом групп клиентов.\
Через кнопку "Добавить" вы можете внести новый тираж, через кнопку "Удалить" ![](<../../../../.gitbook/assets/image (3851).png>) удалить тираж из списка.\
После введения всех данных нажмите "<mark style="background-color:green;">Применить</mark>".

<figure><img src="../../../../.gitbook/assets/image (3850).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Возможно скрыть модуль Тиражность → вариант Выбор из списка на сайте для клиента. Рекомендуется использовать при выборе отображения калькулятора типа "Карточка товара"
{% endhint %}

* **Ручной ввод**

При выборе _Ручного ввода_ продукции на сайте, в настройках вы можете ограничить минимальное количество продукции, максимальное количество и кратность.

![](<../../../../.gitbook/assets/image (2417).png>)

При введении клиентом неверного тиража произойдет установка ближайшего меньшего значения. Например, если установлена кратность 10, при введении значения 62, система подставит 60.

![](<../../../../.gitbook/assets/image (1977).png>)
