### Заголовок на сайте

При заполнении поля *Заголовок на сайте*, для клиента будет видно только выборе варианта ввода количества *Ручной ввод*.

Для подтверждения введенного вручную тиража в карточке товара, нажмите Enter или кликните на любое другое место на странице, тогда введённый тираж применится и рассчитается стоимость.

### Минимальная стоимость

Поле *Минимальная стоимость (в руб.)* при заполнении данного поля, калькуляция не будет считаться ниже введенного значения.

Пример: на сайте продукт Визитки цена за шт. высчитывается исходя из цены печати, бумаги, резки. Но вы не хотите продавать продукт ниже чем за 1000 руб.\
Установив в калькуляции продукта минимальную стоимость 1000 руб. на сайте будет отображать цена 1000 руб. за любой тираж, пока по калькуляции не выйдет более 1000 руб.\

### **Выбор вариантов**

Предусмотрено два варианта внесения количества продукции:

-  **Выбор из списка**

При выборе из списка вы носите тиражи продукции, при этом предусмотрено введение фиксированных цен на весь тираж с учетом групп клиентов.\
Через кнопку "Добавить" вы можете внести новый тираж, через кнопку "Удалить" ![](<../../../../.gitbook/assets/image (3851).png>) удалить тираж из списка.\
После введения всех данных нажмите [highlight:green]"Применить"[/highlight].

![](<../../../../.gitbook/assets/image (3850).png>)

<figcaption>



</figcaption>

:::info 

Возможно скрыть модуль Тиражность -> вариант Выбор из списка на сайте для клиента. Рекомендуется использовать при выборе отображения калькулятора типа "Карточка товара"

:::



-  **Ручной ввод**

При выборе *Ручного ввода* продукции на сайте, в настройках вы можете ограничить минимальное количество продукции, максимальное количество и кратность.

![](<../../../../.gitbook/assets/image (2417).png>)

При введении клиентом неверного тиража произойдет установка ближайшего меньшего значения. Например, если установлена кратность 10, при введении значения 62, система подставит 60.

![](<../../../../.gitbook/assets/image (1977).png>)
