# Верхнее меню

## **Верхнее меню**

Панель управления верхнего меню представляет собой один пункт для редактирования: "Количество пунктов".

<figure><img src="../../../.gitbook/assets/image (3872).png" alt=""><figcaption><p>Общий вид Меню</p></figcaption></figure>

##

## **Количество пунктов**

Нажмите на "Количество пунктов", чтобы изменить пункты меню.

<figure><img src="../../../.gitbook/assets/image (3873).png" alt=""><figcaption></figcaption></figure>

### **Создание нового пункта меню**

Чтобы добавить новый пункт меню, нажмите кнопку "Добавить пункт меню".

![Кнопка "Добавить пункт меню"](<../../../.gitbook/assets/image (2828).png>)

Заполните поля:

_• Заголовок_ — название пункта меню на сайте;

• Тип — вид отображения меню. Подробнее ниже, [Мегаменю](verkhnee-menyu.md#megamenyu);

_• Ссылка_ — ссылка на страницу, которая будет открываться при клике на пункт меню;

{% hint style="warning" %}
В поле Ссылка можно указывать ссылку только на существующую страницу. Если вы хотите добавить какой-либо совершенно новый пункт меню, сначала вам необходимо создать [страницу](https://support.wow2print.com/content/untitled/stranicy/sozdanie-novoi-stranicy).

Если вы хотите чтобы пункт был в меню, но на него не было ссылки на конкретную страницу, в поле ссылка пропишите символ **#**
{% endhint %}

{% hint style="danger" %}
Ссылку необходимо указывать в виде **/shirokoformatnaya-pechat**, избегайте ссылок вида ~~**https://demo2.wow2print.com**~~**/shirokoformatnaya-pechat**
{% endhint %}

_• Родитель_ — выберите родителя, если необходимо данный пункт меню сделать подпунктом. Если это поле оставить пустым, то данный пункт будет отдельным пунктом меню.

![Настройки пункта меню](<../../../.gitbook/assets/image (638).png>)

<br>

### **Подпункт меню**

Пункты меню можно создавать трех уровней.

По рис. "Пункты меню" можно увидеть:

* "Продукция (ручн.)" — первый уровень
  * "Листовки и фирм.бланки" — второй уровень
    * "Листовка А4" — третий уровень
    * "Листовка А5" — третий уровень
    * ...

![Пункты меню](<../../../.gitbook/assets/image (1128).png>)

<br>

### **Редактирование порядка пунктов меню**

Пункты меню "Продукция" и "Дизайн-шаблоны" удалить нельзя, но их можно переименовать, отключить или же передвинуть.

![Инструменты редактирования пунктов меню](<../../../.gitbook/assets/image (2530).png>)

Слева от каждого пункта меню имеется иконка ![](<../../../.gitbook/assets/image (3876).png>), с помощью нее можно перемещать пункты меню между собой. Нажмите, и удерживая левой кнопкой мыши иконку, перетащите пункт меню в нужное положение.

![Сортировка пунктов меню](<../../../.gitbook/assets/image (637).png>)

## **Мегаменю**

Любой пункт меню первого уровня можно отображать, как в стандартном виде, так и в виде Мегаменю.

![Мегаменю (сайт)](<../../../.gitbook/assets/image (663).png>)

Чтобы переключить режим отображения: перейдите в редактирование пункта меню (щелкните мышкой на нужном пункте меню в общем списке), затем в поле _Тип_ выберите вариант _Мегаменю._

![Тип пункта меню](<../../../.gitbook/assets/image (2845).png>)

Есть два вида формирования пунктов в мегаменю: **автоматический** и **ручной.**

![Отображение вида мегаменю](<../../../.gitbook/assets/image (2875).png>)

**Автоматический** — отображает все включенные категории и имеющиеся в них продукты. Вы можете выбрать, какие категории будут отображаться, при помощи бегунков напротив категории.

![Настройки автоматичсекого пункта мегаменю](<../../../.gitbook/assets/image (2526).png>)

{% hint style="danger" %}
При переключении с автоматического режима в ручной, связь пунктов меню и продукции нарушается, то есть, при добавлении или удалении новой продукции, пункты меню обновляться автоматически не будут — только вручную.
{% endhint %}

**Ручной** вариант позволяет выбрать как нужные категории, так и отдельные товары в них.
