## **Редактирование существующих галерей**

При нажатии на название галереи, откроется меню редактирования с тремя вкладками: **Описание**, **Изображения** и **Виджеты**.

В этой вкладке можно изменить только **название галереи**. Название является обязательным полем, которое необходимо заполнить.

#### **Кнопка "Удалить"**

Для удаления изображения, нажмите кнопку "Удалить" ![](<../../.gitbook/assets/image (277).png>)

#### **Кнопка "Изменить"**

Чтобы отредактировать уже загруженные изображения, нажмите кнопку **"Изменить"**. После этого откроется меню редактирования, где вы сможете внести следующие изменения:

**•  Язык:** выбор языковых версий сайта, на которых будет отображаться данное изображение.

**•  Тип устройства:** настройка отображения изображений в зависимости от типа устройства: "Для десктопа" (только для компьютеров), "Для мобильных устройств" (только для смартфонов и планшетов) и "Универсальный" (для всех типов устройств).

**•  Url:** вставьте ссылку в это поле, чтобы сделать изображение кликабельным. При нажатии на изображение на сайте произойдёт переход по указанной ссылке.

•  **Теги:** отметьте изображение тегом и в дальнейшем, при создании виджета, будет возможность сформировались галерею виджета на основе указанного тега. Тег можно ввести самостоятельно либо выбрать из имеющихся.

**•  Alt:** текстовое описание изображения, которое отображается, если изображение не загружено.

**•  Title:** описание изображения, появляющееся при наведении курсора.

**•  Видео:** возможность загрузки видеофайлов (доступно только для виджета "Слайдер").

:::note 

**Важно! Видео отображается без звука и только при автоматической смене изображений.**

:::

### **Создание кнопок на слайдере**

На каждом изображении в галереи можно создать кнопки для быстрого перехода по ссылке, либо для открытия формы обратной связи.

Для создания кнопки установите переключатель "Кнопка на слайдере"  в положение "Включен" и заполните открывшиеся поля:

**•  Заголовок:** это основное наименование темы над кнопкой слайда.

**•  Описание:** краткое описание слайда, которое отображается под заголовком.

**•  Стиль заголовка / Стиль описания:** позволяет выбрать стиль текста. Подробнее в статье [Типографическая система](./../../settings/dizain-saita/tipograficheskaya-sistema#tekst).

**•  Цвет заголовка / Цвет описания:** позволяет подобрать цвет из палитры для заголовка и описания

**•  Подпись кнопки:** название кнопки, указывающее о действии при нажатии

**•  Позиция модуля:** позволяет выбрать расположение всего модуля (Заголовок, Описание, Кнопка) на слайде . Доступны три положения: слева, по центру и справа

**•  Цель:** выбор действия кнопки (открытие формы обратной связи или переход по ссылке).

**•  Ссылка:** поле для ввода ссылки, если выбрана соответствующая цель.

#### **Импорт настроек кнопки**

Чтобы не настраивать кнопки для каждого изображения вручную, используйте функцию "Импорт настроек кнопки".

При нажатии откроется окно "Импорт настроек". В данном окне выберите на какие изображения будет скопирована настройка кнопки.

## **Создание новой Галереи**

Изображения автоматически попадают в раздел "Галерея" при загрузке в виджетах "О типографии", "Слайдер", "Фото-карусель" и "Логотипы".

В разделе **"Галереи"** предусмотрена возможность добавить свою галерею. Создать её можно при нажатии кнопки **"Добавить"**.

<figure>

![](<../../.gitbook/assets/image (287).png>)

<figcaption>



</figcaption>

</figure>

После нажатия кнопки, откроется страница создания галереи. Введите название и нажмите **"Сохранить"**.

<figure>

![](<../../.gitbook/assets/image (288).png>)

<figcaption>



</figcaption>

</figure>

Для загрузки изображений, перейдите во вкладку "**Изображения"** и нажмите кнопку "**Загрузить"**. После загрузки изображение можно отредактировать или удалить.
