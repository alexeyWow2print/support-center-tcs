Раздел Файлы используется для загрузки специальных файлов, например, для подтверждения почты для домена. 

:::info 

Для подтверждения права на сайт необходимо скачать html-файл в кабинете соответствующего сервиса (например, Яндекс.Вебмастер или Google Search Console) и загрузить его в данном разделе, нажав на кнопку "Добавить". 

При добавлении файла необходимо указать на него url. Url имеет вид названия файла. 

Например, название файла yandex_959c5d0fb15dc10f6.html, его и нужно вставить в поле url. Не забудьте после этого включить файл на сайте.

:::



#### 

#### **Добавление файла**

Чтобы добавить файл нажмите кнопку "Добавить"

![](<../../.gitbook/assets/image (556).png>)



Пропишите url файла и загрузите необходимый файл через соответствующую кнопку "Файл" -> "Сохранить".

![](<../../.gitbook/assets/image (1320).png>)



:::note 

**Важно!** Чтобы файл отображался на сайте, в конце url необходимо прописать формат загружаемого файла: .jpg, .png, .svg и т.д..

:::





#### Отображение файла на сайте/Удаление файла

![](<../../.gitbook/assets/image (3144).png>)

При помощи переключателя "Выкл/Вкл" можно отобразить или скрыть отображения на сайте.

Чтобы посмотреть файл или удалить воспользуйтесь кнопками напротив загруженного файла.



#### Структурирование файлов по папкам

Файлы можно структурировать по папкам. Чтобы создать папку нажмите на кнопку "Добавить папку" и введите название:

![](<../../.gitbook/assets/image (2330).png>)



Чтобы разместить файл в папке выберите нужную папку в файле.

![](<../../.gitbook/assets/image (1687).png>)
