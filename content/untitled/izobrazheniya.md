# Изображения

### Создание папок

В разделе предусмотрено формирование папок и дальнейшей загрузки в них файлов.

![](<../../.gitbook/assets/image (3643).png>)



Чтобы создать папку нажмите на кнопку "Добавить" в правом верхнем углу и заполните поле _Название_ в открывшейся вкладке "Описание".

![](<../../.gitbook/assets/image (1182).png>)



Для удобства каталогизации изображений предусмотрена возможность создавать папки внутри папок.

{% hint style="warning" %}
Нельзя создать "подпапку" в папке, где уже есть изображения.
{% endhint %}

Для того, чтобы "положить" папку одну в другую, выберите нужную в выпадающем списке в поле _Папка_. В списке будут отображаться только пустые папки, не содержащие изображения.

![](<../../.gitbook/assets/image (806).png>)



Вы также можете свободно перемещать папки из одной в другую, выбрав во вкладке Описание нужную папку из списка.

<figure><img src="../../.gitbook/assets/image (270).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Ссылка изображения при этом не изменится.
{% endhint %}





### Загрузка файлов в папки

После сохранения Описания, во вкладке "Изображения" вы можете загрузить необходимые для работы картинки, используя кнопку "Загрузить".

<figure><img src="../../.gitbook/assets/image (3879).png" alt=""><figcaption></figcaption></figure>

**Типы поддерживаемых изображений:** .jpg, .jpeg, .gif, .png, .webp, .svg.

**Минимальный размер:** 643х300 px

&#x20;**Размер:** до 10MB.



### **Удаление файлов из папки**

После загрузки изображений, они сохраняются во вкладку "Изображения", где вы можете их удалит&#x44C;**,** нажав на кнопку "Удалить" ![](<../../.gitbook/assets/image (271).png>).

<figure><img src="../../.gitbook/assets/image (272).png" alt=""><figcaption></figcaption></figure>





## **Алгоритм использования загруженных изображений**

**Контент.** Загруженные изображения вы можете использовать при формировании контента на странице. Для этого, в редактируемом поле страницы, нажмите на иконку "Изображение".

![](<../../.gitbook/assets/image (3059).png>)

В окне появится возможность выбрать файл с компьютера или выбрать из загруженных изображений.

![](<../../.gitbook/assets/image (3238).png>)



### **Ссылка для верстки**

Для удобства верстки страниц сайта на изображения в папках можно получить ссылку для использования ее в коде. для этого во вкладке "Изображения" нажмите на кнопку "Скопировать ссылку".

<figure><img src="../../.gitbook/assets/image (273).png" alt=""><figcaption></figcaption></figure>
