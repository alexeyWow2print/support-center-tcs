## Как создать?

Чтобы создать виджет «Форма обратной связи», в админ-панели сайта войдите в раздел «_Контент → Виджеты»_, нажмите на кнопку «Добавить» в правом верхнем углу. В открывшемся окне найдите виджет «Форма обратной связи» и нажмите «Создать».





## Параметры

###

### Общие

Перед вами откроется форма с возможностью выбрать параметры виджета.

![](<../../.gitbook/assets/image (843).png>)



Заполните поля и выберите параметры:

* **Название** виджета\
  Внутреннее название для админ-панели. Нигде не отображается;<br>
* **Название формы на сайте**\
  Добавляет заголовок типа H3, который отображается над виджетом;<br>
* **Подпись**\
  Добавляет текст под название виджета типа основной наборный текст (толщина 1);<br>
* **Подпись кнопки**\
  Добавляет текст на кнопку.





### Дополнительные

После сохранения настроек виджета, отобразятся новые параметры.

![](<../../.gitbook/assets/image (1044).png>)



Здесь можно настроить отображение полей Формы. Каждое поле имеет следующие параметры:

* **Наименование**\
  Название формы на сайте;<br>
* **Тип** поля
  * Текстовое поле — поле для ввода текста одной строкой;
  * Большое текстовое поле — поле для ввода текста в несколько строк;
  * Файл — поле для загрузки файла.<br>
* **Фильтр**\
  В качестве чего будет восприниматься информация из этого поля. Данный параметр важен при получении обращения от клиента в админ-панели сайта;<br>
* **Отображать на сайте**\
  Отображать или не отображать данное поле на сайте;<br>
* **Обязателен**\
  Параметр нельзя оставить не заполненным;<br>
* **Маска**\
  Маска для номера телефона в Форме, чтобы клиенту было удобнее заполнять номер телефона.
  * +9 (999) 99?9-99999999 — универсальная маска;
  * +7 (999) 999-99-99 — для номеров РФ;
  * +380 (99) 999-99-99 — для номеров Украины;
  * +373(99) 99-99-99 — для номеров Молдавии;
  * +999999?999999999 — свободная форма.



{% hint style="warning" %}
Не забудьте активировать виджет после создания. Это можно сделать в разделе «Контент → Виджеты», путем переключения бегунка в состояние Вкл.
{% endhint %}

## Порядок установки (2 вар.)

###

### 1 вариант — Через вставку кода

После сохранения всех параметров, скопируйте «Код для установки на сайт».

![](<../../.gitbook/assets/image (3407).png>)



Перейдите на нужную страницу или продукт, в режиме исходного кода вставьте код виджета в то место, которое необходимо.\
Готово!

![](<../../.gitbook/assets/Вставка кода 1.0 (2).gif>)





### 2 вариант — Через редактор страниц

Перейдите в раздел "Контент → Наполнение сайта → Страницы" нажмите на название страницы. Вы окажитесь в редакторе страниц.\
Слева выберите необходимый виджет и вставьте в поле правее в нужном порядке.\
Готово!

![](<../../.gitbook/assets/Виджет в конструкторе страниц (1).gif>)
