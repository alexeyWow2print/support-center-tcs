## Как создать?

Чтобы создать виджет «Преимущества», в админ-панели сайта войдите в раздел «_Контент → Виджеты»_, нажмите на кнопку «Добавить» в правом верхнем углу. В открывшемся окне найдите виджет «Преимущества» и нажмите «Создать».





## Параметры

###

### Общие

Перед вами откроется форма с возможностью выбрать параметры виджета.

![](<../../.gitbook/assets/image (1317).png>)



Заполните поля и выберите параметры:

* **Название** виджета\
  Внутреннее название для админ-панели. Нигде не отображается.<br>
* **Тип устройства**
  * Универсальный — виджет будет отображаться на всех устройствах;
  * Для десктопа — отображение будет только на компьютере/ноутбуке;
  * Для мобильных устройств — отображение только на мобильных устройствах.<br>
* **Количество в ряд**\
  Можно выбрать 3, 4 или 6 элементов в ряд.<br>
* **Текст**\
  Расположение заголовка элемента:
  * Снизу — заголовок и описание элемента будут располагаться под изображением;
  * Справа — заголовок и описание элемента будут располагаться справа от изображения.

Далее идут настройки элементов, каждый элемент имеет индивидуальные настройки:

* **Заголовок**\
  Заголовок элемента типа основной наборный текст (толщина 3).<br>
*   **Описание**\
    Текст описания элемента типа малый наборный текст.


* **Url**\
  При клике на изображение элемента, клиент будет переадресован на указанный url.<br>
* **Изображение**\
  В каждый элемент необходимо загрузить соответствующее изображение.<br>
* **Добавить**\
  Количество элементов не ограничено, с помощью кнопки «Добавить», в виджет можно добавить любое количество элементов.



{% hint style="warning" %}
Не забудьте активировать виджет после создания. Это можно сделать в разделе «Контент → Виджеты», путем переключения бегунка в состояние Вкл.
{% endhint %}





### Требования к изображениям

Требования к изображениям не зависят от параметров виджета.

**Размер изображения**:\
120 x 120 px.

**Допустимые форматы**:\
.jpeg, .png и .gif.





## Порядок установки (2 вар.)

###

### 1 вариант — Через вставку кода

После сохранения всех параметров, скопируйте «Код для установки на сайт».

![](<../../.gitbook/assets/image (3407).png>)



Перейдите на нужную страницу или продукт, в режиме исходного кода вставьте код виджета в то место, которое необходимо.\
Готово!

![](<../../.gitbook/assets/Вставка кода 1.0 (2).gif>)





### 2 вариант — Через редактор страниц

Перейдите в раздел "Контент → Наполнение сайта → Страницы" нажмите на название страницы. Вы окажитесь в редакторе страниц.\
Слева выберите необходимый виджет и вставьте в поле правее в нужном порядке.\
Готово!

![](<../../.gitbook/assets/Виджет в конструкторе страниц (1).gif>)
