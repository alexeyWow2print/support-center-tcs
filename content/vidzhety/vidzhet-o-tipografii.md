## Как создать?

Чтобы создать виджет «О типографии», в админ-панели сайта войдите в раздел «_Контент → Виджеты»_, нажмите на кнопку «Добавить» в правом верхнем углу. В открывшемся окне найдите виджет «О типографии» и нажмите «Создать».





## Параметры

###

### Общие

Перед вами откроется форма с возможностью выбрать параметры виджета.

![](<../../.gitbook/assets/image (834).png>)



Заполните поля и выберите параметры:

* **Название** виджета\
  Внутреннее название для админ-панели. Нигде не отображается.<br>
* **Заголовок**\
  Добавляет заголовок типа H2, который отображается над виджетом.<br>
* **Описание**\
  Добавляет в виджет текст описания, который отображается в правой колонке.<br>
* **Текст ссылки**\
  Текст ссылки.<br>
* **Url ссылки**\
  При клике на ссылку, клиент будет переадресован на соответствующую страницу. В этом поле можно указать ссылку, например, на полноценную страницу сайта «О нас».<br>
* **Дата основания**\
  Дата основания будет отображаться крупным шрифтом в верхней части виджета.<br>
* **Количество выполненных заказов**\
  Число, которое показывает количество выполненных заказов.<br>
* **Тип устройства**
  * Универсальный — виджет будет отображаться на всех устройствах;
  * Для десктопа — отображение будет только на компьютере/ноутбуке;
  * Для мобильных устройств — отображение только на мобильных устройствах.



### Дополнительные

После сохранения настроек, появится новая вкладка «Изображения». Перейдите на эту вкладку и загрузите изображения типографии.\
Изображения будут отображаться в левой части виджета в виде слайдера.





### Требования к изображениям

**Размер изображения**:\
560 x 400 px

**Допустимые форматы**:\
.jpeg, .png и .gif



{% hint style="warning" %}
Не забудьте активировать виджет после создания. Это можно сделать в разделе «Контент → Виджеты», путем переключения бегунка в состояние Вкл.
{% endhint %}





## Порядок установки (2 вар.)

###

### 1 вариант — Через вставку кода

После сохранения всех параметров, скопируйте «Код для установки на сайт».

![](<../../.gitbook/assets/image (3407).png>)



Перейдите на нужную страницу или продукт, в режиме исходного кода вставьте код виджета в то место, которое необходимо.\
Готово!

![](<../../.gitbook/assets/Вставка кода 1.0 (2).gif>)





### 2 вариант — Через редактор страниц

Перейдите в раздел "Контент → Наполнение сайта → Страницы" нажмите на название страницы. Вы окажитесь в редакторе страниц.\
Слева выберите необходимый виджет и вставьте в поле правее в нужном порядке.\
Готово!

![](<../../.gitbook/assets/Виджет в конструкторе страниц (1).gif>)
