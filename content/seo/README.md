# SEO

## Общие положения

_SEO (Search Engine Optimization) – это комплекс мер по улучшению сайта для его ранжирования в поисковых системах._\
&#xNAN;_&#x53;EO настройка сайта — это внутренняя оптимизация страниц сайта для продвижения в поиске. Поисковые системы работают по заданным алгоритмам, а SEO-настройки помогают поисковым роботам глубже понять смысл и ценность контента._

{% hint style="info" %}
Если вам необходимо настроить мета-теги страницы (title, description, keywords и canonical link), то их необходимо настраивать в настройках конкретной страницы сайта в разделе [страницы](../untitled/stranicy/) или [продукция](../../product/).\
У каждой страницы сайта имеется вкладка SEO с соответствующими настройками.
{% endhint %}
