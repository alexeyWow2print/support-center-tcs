В wep-to-print сервисе TCS карта сайта представлена в виде файла Sitemap.xml, который расположен в корне сайта и содержит данные о его страницах.\
Поисковые роботы Яндекс и Google заходят на сайт и «гуляют» по нему какое-то время. Чтобы у робота был четкий путь и он не индексировал все подряд, помогает карта сайта.

Карта сайта выглядит следующим образом:

```
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
<url>
<loc>https://demo.wow2print.com</loc>
<priority>1.0</priority>
</url>
<url>
<loc>https://demo.wow2print.com/delivery</loc>
<priority>0.5</priority>
</url>
...
</urlset>
```



В данном подразделе можно управлять картой сайта.

![](<../../.gitbook/assets/image (3530).png>)



В левой части отображается информация о последнем обновлении файла карты сайта.\
В правой части кнопки для взаимодействия с картой сайта:

-  **Обновить карту сайта**\
   Данная кнопка позволяет принудительно обновить файл карты сайта.

-  **Скачать карту сайта**\
   Карту сайта можно редактировать вручную, для этого вы можете ее создать через кнопку «Обновить карту сайта» и скачать файл с помощью следующей кнопки.

-  **Загрузить карту сайта**\
   После того, как вы вручную внесли изменения в файл карты сайта, его можно загрузить обратно.

:::note 

Если после ручного редактирования файла принудительно обновить карту сайта в админ-панели, то все изменения сбросятся и карта сайта будет изменена по тем правилам, которые заложены в код сайта.

:::
