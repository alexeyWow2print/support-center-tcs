# 301 редирект

При работе с сайтом могут возникнуть ситуации, когда происходит удаление, перенос или изменение url-адреса отдельных страниц или сайта целиком. При этом требуется сохранить индексацию в поисковых системах и перенаправить пользовательский трафик на новый адрес.\
С данной задачей поможет справится 301 редирект, то есть переадресация с одного url адреса на другой.

Чтобы добавить переадресацию, нажмите кнопку «Добавить» в правом верхнем углу окна браузера, перед вами откроется окно следующего вида:

![](<../../.gitbook/assets/image (2249).png>)



* **Проверяемый url**\
  Url с которого происходит переадресация.\
  Можно настроить переадресацию только со страниц текущего сайта. Нельзя указать проверяемый url с другого ресурса.\
  Ссылка должна иметь относительный вид, то есть в ней не должен содержаться адрес самого сайта, только то что находится после символа «/». \
  Например, «produkciya/buklety-win.html» (вместо «https://demo.wow2print.com/produkciya/buklety-win.html»)<br>
* **Переадресация на**\
  Url на который происходит переадресация.\
  Переадресация может происходить на любой другой ресурс.\
  Если переадресация происходит на сторонний ресурс, то ссылка должна иметь абсолютный вид — url целиком.\
  Например, «https://www.youtube.com/channel/UCo1XtkacdqjSkiB-40DVhug»<br>
* **Статус**\
  Статус редиректа — включен или выключен
