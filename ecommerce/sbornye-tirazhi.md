# Тиражи

### Условия формирования тиража

В Настройки → Статусы → Статусы заказов → Статусы выберите статус на котором предполагается создание тиража

<figure><img src="../.gitbook/assets/Screenshot_2 (4).png" alt=""><figcaption><p><em>Рис. 1 - Выбор статуса заказа для создания тиража</em></p></figcaption></figure>

{% hint style="warning" %}
#### <mark style="color:red;">Помните:</mark>&#x20;

Формирование Тиража возможно только при соблюдении следующих требований:

1. Заказ должен быть оплачен или установлена гарантия оплаты
2. Макет продукта загружен и принят
{% endhint %}





### Создание тиража

Для создания тиража зайдите в Магазин → Заказы → Товары.&#x20;

Отметьте те товары, макеты которых вы хотите сформировать в тираж, и нажмите кнопку «Создать тираж».

<figure><img src="../.gitbook/assets/Screenshot_12 (7).png" alt=""><figcaption><p><em>Рис. 2 - Порядок действий для создания тиража</em></p></figcaption></figure>

Выберите «Создать новый», введите название тиража и нажмите «Создать»

<figure><img src="../.gitbook/assets/Screenshot_6 (11).png" alt=""><figcaption><p><em>Рис. 3 - Создание нового тиража</em></p></figcaption></figure>



### Добавление нового макета в уже существующий тираж

Вы можете не только создавать новые тиражи, но и добавлять макеты к уже существующим тиражам.

Для этого в Магазин → Заказы → Товары выберите нужный товар → нажмите кнопку «Создать тираж» - и выберите из выпадающего списка тираж, к которому хотите добавить товары.

&#x20;Макет добавится к указанному тиражу.

<figure><img src="../.gitbook/assets/Screenshot_7 (7).png" alt=""><figcaption><p><em>Рис. 4 - Добавление товаров к существующему тиражу</em></p></figcaption></figure>



### Работа с тиражами

В Магазин → Тиражи вы найдете все сформированные тиражи.

<figure><img src="../.gitbook/assets/Screenshot_1 (25).png" alt=""><figcaption><p><em>Рис. 5 - Список тиражей отображает все созданные тиражи</em></p></figcaption></figure>

В каждом тираже отображается превью макетов товаров.

Также предусмотрен быстрый доступ к товарам по номеру товара (например на скрине, №82, 83).&#x20;

Скачать макеты всех товаров из тиража в исходном формате можно нажав на кнопку «Макеты» → «Исходные».&#x20;

Если вы используете проверку макетов на заданные параметры, то по кнопке «Макеты TCS» вы можете скачать макеты в формате PDF.

По кнопке «Скачать в файл» формируется список всех товаров с деталями из тиража (в формате ecxel).

<figure><img src="../.gitbook/assets/Screenshot_5 (6).png" alt=""><figcaption><p><em>Рис. 6 - Работа с тиражами</em></p></figcaption></figure>



### Статусы тиражей

Изменять статусы тиражей можно в панели управления внизу раздела, отметив тираж и выбрав нужный статус.&#x20;

Тираж переместится во вкладку, сопоставленную со статусом. При этом товары, добавленные в тираж, также изменят статус.

<figure><img src="../.gitbook/assets/Screenshot_4 (23).png" alt=""><figcaption><p><em>Рис. 7 - Изменение статуса тиража</em></p></figcaption></figure>



Также изменить статус тиража можно непосредственно в самом тираже в поле Статус тиража (рис. 6)

Добавить, переименовать или удалить статусы тиражей и вкладки вы можете в Настройки → Статусы → Статусы тиражей.&#x20;

Здесь же вы можете сопоставить статус тиража со статусом товара. В случае сопоставления, при изменении статуса тиража, атовматически изменится и статус товара.

<figure><img src="../.gitbook/assets/Screenshot_3 (23).png" alt=""><figcaption><p><em>Рис. 8 - Настройки статусов тиражей</em></p></figcaption></figure>





### Удаление тиражей

Удалить тиражи вы можете, выбрав их из списка и нажав на значок "Корзина".
