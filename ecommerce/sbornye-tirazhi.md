Модуль Тиражи позволяет объединить макеты разных продуктов в один архив для дальнейшей работы с ними.



### Условия формирования тиража

В Настройки -> Статусы -> Статусы заказов -> Статусы выберите статус на котором предполагается создание тиража

<figure>

![](<../.gitbook/assets/Screenshot_2 (4).png>)

<figcaption>

*Рис. 1 - Выбор статуса заказа для создания тиража*

</figcaption>

</figure>



:::tip 

#### <mark style="color:red;">Помните:</mark> 

Формирование Тиража возможно только при соблюдении следующих требований:

1. Заказ должен быть оплачен или установлена гарантия оплаты

2. Макет продукта загружен и принят

:::







### Создание тиража

Для создания тиража зайдите в Магазин -> Заказы -> Товары. 

Отметьте те товары, макеты которых вы хотите сформировать в тираж, и нажмите кнопку «Создать тираж».

<figure>

![](<../.gitbook/assets/Screenshot_12 (7).png>)

<figcaption>

*Рис. 2 - Порядок действий для создания тиража*

</figcaption>

</figure>

Выберите «Создать новый», введите название тиража и нажмите «Создать»

<figure>

![](<../.gitbook/assets/Screenshot_6 (11).png>)

<figcaption>

*Рис. 3 - Создание нового тиража*

</figcaption>

</figure>



### Добавление нового макета в уже существующий тираж

Вы можете не только создавать новые тиражи, но и добавлять макеты к уже существующим тиражам.

Для этого в Магазин -> Заказы -> Товары выберите нужный товар -> нажмите кнопку «Создать тираж» - и выберите из выпадающего списка тираж, к которому хотите добавить товары.

 Макет добавится к указанному тиражу.

### Работа с тиражами

В Магазин -> Тиражи вы найдете все сформированные тиражи.

<figure>

![](<../.gitbook/assets/Screenshot_1 (25).png>)

<figcaption>

*Рис. 5 - Список тиражей отображает все созданные тиражи*

</figcaption>

</figure>

В каждом тираже отображается превью макетов товаров.

Также предусмотрен быстрый доступ к товарам по номеру товара (например на скрине, №82, 83). 

Скачать макеты всех товаров из тиража в исходном формате можно нажав на кнопку «Макеты» -> «Исходные». 

Если вы используете проверку макетов на заданные параметры, то по кнопке «Макеты TCS» вы можете скачать макеты в формате PDF.

По кнопке «Скачать в файл» формируется список всех товаров с деталями из тиража (в формате ecxel).

### Статусы тиражей

Изменять статусы тиражей можно в панели управления внизу раздела, отметив тираж и выбрав нужный статус. 

Тираж переместится во вкладку, сопоставленную со статусом. При этом товары, добавленные в тираж, также изменят статус.

<figure>

![](<../.gitbook/assets/Screenshot_4 (23).png>)

<figcaption>

*Рис. 7 - Изменение статуса тиража*

</figcaption>

</figure>



Также изменить статус тиража можно непосредственно в самом тираже в поле Статус тиража (рис. 6)

Добавить, переименовать или удалить статусы тиражей и вкладки вы можете в Настройки -> Статусы -> Статусы тиражей. 

Здесь же вы можете сопоставить статус тиража со статусом товара. В случае сопоставления, при изменении статуса тиража, атовматически изменится и статус товара.

</figure>
