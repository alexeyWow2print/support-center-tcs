Модуль Отгрузки позволяет сформировать в одну отгрузку товары из разных заказов, при условии что это заказы одного клиента.

### Список отгрузок

В списке отгрузок отображаются по вкладкам все созданные отгрузки

![](<../.gitbook/assets/image (3295).png>)

### 

### Статусы и вкладки отгрузок

Настроить статусы отгрузок, создать вкладки и сопоставить их между собой, вы можете в разделе

:::quote 

Настройки -> Статусы -> Статусы отгрузок

:::



![](<../.gitbook/assets/image (1253).png>)

Изменять статусы отгрузок и перемещать по вкладкам можно в панели управления внизу раздела, отметив  отгрузку и выбрав нужный статус. Отгрузка переместится во вкладку сопоставленную со статусом. 

![](<../.gitbook/assets/image (1035).png>)





### Формирование отгрузки 

Для формирование отгрузки в Магазин -> Товары -> выберите нужные позиции товара и нажмите "Создать отгрузку". 

![](<../.gitbook/assets/image (1348).png>)



Впишите название отгрузки и выберите Получателя из списка предложенных (если у клиента много получателей).

![](<../.gitbook/assets/image (1199).png>)



Выберите (подтвердите) способ доставки или пункт самовывоза -> нажмите кнопку "Создать"

![](<../.gitbook/assets/image (880).png>)





После нажатия кнопки "Создать" вы автоматически переходите на страницу отгрузки, где будут указана информация о клиенте, получателе, параметрах, товарах. В правом верхнем углу вы можете распечатать отгрузочный лист.

![](<../.gitbook/assets/image (2869).png>)



пример отгрузочного листа:

![](<../.gitbook/assets/image (3085).png>)



Созданная отгрузка отобразится в разделе

:::quote 

Магазин -> Отгрузки-> вкладка Новый

:::

Статус Новый и Завершенный стоят по умолчанию и не подлежат удалению. Для удобства вы можете переименовать их или назначить дополнительные статусы и создать вкладки. Подробнее от этом в разделе [Справочника](./../handbook/_index).



### Закрывающие документы

Формирование закрывающих документов можно настроить на удобном для вас статусе в 

:::quote 

Настройки -> Статусы -> Статусы отгрузок -> Закрывающие документы (отметить)

:::

![](<../.gitbook/assets/image (1004).png>)



Когда отгрузка получит статус на котом формируются закрывающие документы, эти документы можно скачать в 

:::quote 

Магазин -> Бухгалтерия -> Реализация

:::

![](<../.gitbook/assets/image (3086).png>)



:::info 

 Внимание! Дату закрывающих документов вы можете корректировать.

:::



или в Магазин -> Отгрузки -> нужная отгрузка-> кнопка "Документы"

![](<../.gitbook/assets/image (1019).png>)
