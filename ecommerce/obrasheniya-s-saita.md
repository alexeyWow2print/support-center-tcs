# Обращения с сайта

###

### Список обращений&#x20;

В разделе Магазин → Обращения с сайта вы можете управлять обращениями с сайта, полученными через формы обратной связи&#x20;

> Настройки → Сайт → Формы обратной связи



![](<../.gitbook/assets/image (1985).png>)



Обращения от новых, ранее не зарегистрированных пользователей попадают во вкладку Неразобранные.

![](<../.gitbook/assets/image (2611).png>)

&#x20;

Чтобы закрепить обращение за менеджером (пользователем, имеющим доступ "Работа с обращениями с сайта") нажмите кнопку "Принять".&#x20;

![](<../.gitbook/assets/image (1017).png>)





{% hint style="info" %}
При поступлении обращения с сайта, система осуществляет клиента по email. Если клиент зарегистрирован на сайте, то обращение автоматически закрепляется за клиентом.
{% endhint %}



### Статусы и вкладки&#x20;

Создать статусы, вкладки и настроить их сопоставление можно в

> Настройки → Статусы → Статусы обращений с сайта



Изменять статусы обращений и перемещать их по вкладкам можно в панели управления внизу раздела, отметив обращение и выбрав нужный статус. Обращение переместится во вкладку сопоставленную со статусом.&#x20;

![](<../.gitbook/assets/image (507).png>)





### **Удаление обращений**&#x20;

Удалить обращения вы можете выбрав их из списка и нажав на значок "Корзина".&#x20;

![](<../.gitbook/assets/image (1112).png>)



Удаленные обращения автоматически попадают в раздел Помеченные на удаление.&#x20;

![](<../.gitbook/assets/image (1921).png>)





### Создание заказа&#x20;

Системой предусмотрена возможность создания заказа из обращения через кнопку "Создать заказ"  напротив каждого обращения.&#x20;

![](<../.gitbook/assets/image (2473).png>)



Нажав ее, вы попадаете в "Калькуляцию в админ-панели", где вы можете создать заказ самостоятельно. Выберите слева из списка  продукции нужный,  подберите параметры и нажмите "Добавить".

![](<../.gitbook/assets/image (795).png>)



В калькуляции предусмотрена возможность выбрать клиента на этапе подбора параметров. В случае, если клиент, зарегистрирован в системе, будет применена скидка для этого клиента и подтянется группа, если она есть.

После добавления товара вы сможете продолжить оформление заказа, нажав на кнопку "Оформить заказ" в Корзине.



![](<../.gitbook/assets/image (1712).png>)

В Корзине  выполните необходимые действия:&#x20;

![](<../.gitbook/assets/image (949).png>)

* загрузите макет (при наличии) в окне слева&#x20;
* если клиент не был выбран ранее, через строку поиска вы можете добавить существующего клиента или добавьте нового через кнопку "Добавить клиента". При добавлении нового клиента, в форму автоматически подставляются контактные данные из формы обратной связи: Имя и e-mail.
*   после указания/добавления клиента откроется возможность выбора всех остальных параметров Корзины, необходимых для заказа:

    * города клиента (для расчета доставки),&#x20;
    * предлагаемых на сайте способов доставки
    * добавления Плательщика и адреса,
    * вариантов оплаты (на физ лицо или юр. лицо)
    * способов оплаты на сайте (платежные системы, по счету, наличный платеж и т.д.)

    \
    После создания заказа он отобразится в общем списке заказов в&#x20;

> Магазин → Заказы

###

### Создание коммерческого предложения

Кроме создания заказа, в Корзине предусмотрена генерация коммерческого предложения на основании калькуляции заказа через кнопку "Коммерческое предложение".

![](<../.gitbook/assets/image (3353).png>)

\
В открывшемся окне вы можете редактировать цену, выбрать компанию от которой будет выставлено предложение, скачать или отправить коммерческое по указанной  эл. почте.

![](<../.gitbook/assets/image (773).png>)
