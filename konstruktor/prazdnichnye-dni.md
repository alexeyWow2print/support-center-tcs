# Праздничные дни

## **Общая информация**

Данный раздел предназначен для настройки отображения праздничных и выходных дней для календарной сетки в редакторе фотокалендаря. Когда вы отмечаете определенные даты как праздничные, они автоматически выделяются в сетке календаря, что позволяет клиентам видеть нерабочие дни.

<figure><img src="../.gitbook/assets/image (243).png" alt=""><figcaption></figcaption></figure>

Праздники объединяются в группы, что дает возможность настраивать календари для разных целей. Например, можно создать отдельные группы для государственных праздников, православных или региональных памятных дат.

<figure><img src="../.gitbook/assets/image (244).png" alt=""><figcaption></figcaption></figure>



## **Создание новой группы праздников**

Чтобы создать группу, выполните следующие действия:

1. Нажмите кнопку **"Создать"**, расположенную в правом верхнем углу экрана.
2. В открывшемся окне **"Создать группу праздников"** введите название группы (например, "Государственные праздники 2025").
3. Нажмите кнопку **"Сохранить"** для завершения процесса.

<figure><img src="../.gitbook/assets/image (3933).png" alt=""><figcaption></figcaption></figure>



## **Добавление праздничных дней**

Для добавления новых праздников:

1. Откройте нужную группу праздников из общего списка.
2. Нажмите кнопку **"Добавить праздник"**.
3. Заполните следующие поля:
   * **Название** – укажите название праздника (например, "Новый год").
   * **Дата** – выберите конкретную дату, которая должна отмечаться как праздничный день. Обратите внимание, что система не поддерживает ввод диапазона дат – каждая дата добавляется отдельно.
   * **"Каждый год в выбранную дату"** – активируйте эту опцию, если праздник является ежегодным (например, 1 января). В этом случае система будет автоматически отмечать эту дату как праздничную во всех последующих годах.
4. Нажмите **"Сохранить"**, чтобы добавить праздничный день.

<figure><img src="../.gitbook/assets/image (3934).png" alt=""><figcaption></figcaption></figure>

### **Применение группы праздников для фотокалендаря**

Чтобы выбранная группа праздников отображалась в фотокалендаре воспользуйтесь одним из двух способов:

#### **Способ 1: Через раздел "Стили календарей"**

1. Перейдите в **Конструктор → Стили календарей**.
2. Нажмите кнопку **"Создать"** или выберите существующий стиль для редактирования.
3. Внизу страницы найдите параметр **"Группа праздников"**.
4. Выберите нужную группу из раскрывающегося списка.
5. Сохраните изменения.

Подробнее о настройке стилей календарей вы можете узнать в статье ["Стили календарей"](stili-kalendarei.md)

#### **Способ 2: При создании калькуляции**

1. При создании калькуляции добавьте операцию **"Стили календарей"** (добавить конструкторский размер календаря, после в списке добавляемых модулей появится эта операция).
2. В операции нажмите кнопку **"Добавить"**, затем выберите **"Создать"**.
3. В открывшейся форме найдите параметр **"Группа праздников"** он расположен в нижней части окна.
4. В раскрывающемся списке выберите нужную группу праздников.
5. Сохраните изменения.

<figure><img src="../.gitbook/assets/image (247).png" alt=""><figcaption></figcaption></figure>

После выполнения этих действий в календаре будут отображаться все праздничные дни, включенные в выбранную группу.



### **Удаление праздников**

&#x20;Для удаления праздника нажмите на значок корзины ![](<../.gitbook/assets/image (249).png>) в строке с соответствующей датой и подтвердите действие.
