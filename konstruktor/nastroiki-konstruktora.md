# Настройки конструктора

В данном разделе все настройки отвечают за возможность тем или иным способом загрузить изображения в конструктор.

![Настройки конструктора. Загрузка изображений](<../.gitbook/assets/image (1393).png>)

## Социальные сети

### Vkontakte (Вконтакте)

Для выгрузки фотографий из социальной сети Вконтакте, необходимо создать соответствующее приложение и внести ключи этого приложения в настройки сайта.

Подробная инструкция по созданию приложения в Вконтакте: [Авторизация через соцсети](../settings/drugie-nastroiki/avtorizaciya-cherez-socseti/).

После создания приложения, необходимо отметить пункт "Vkontakte" и, в поле "Приложение для Vkontakte", разместить ID приложения.

![Приложение для Vkontakte](<../.gitbook/assets/image (974).png>)

### Facebook

Для выгрузки фотографий из социальной сети Facebook, необходимо создать соответствующее приложение и внести ключи этого приложения в настройки сайта.

После создания приложения, необходимо отметить пункт "Facebook" и, в поле "Приложение для Facebook", разместить ID приложения.

![Приложение для Facebook](<../.gitbook/assets/image (2786).png>)

## Фотобанк

Отметьте данный пункт, если хотите дать возможность клиенту выбирать изображения из вашего каталога изображений. Фотобанк пополняется и настраивается здесь: [Фотобанк](fotobank.md)

## Проверка заполнения проекта

Проверка заполнения проекта предназначена для проверки заполнения компонентов "Изображение" в проекте клиента. При его активации появляется поле ввода, в котором необходимо указать допустимый процент заполнения проекта.

![Проверка заполнения проекта](<../.gitbook/assets/image (1365).png>)

Если клиент заполнил свой проект меньше чем на 95% от общего числа компонентов "Изображение", то при попытке добавить проект в корзину, выйдет следующее предупреждение:

## Инстаграм (Instagram)

Для выгрузки фотографий из социальной сети Instagram, необходимо создать соответствующее приложение на developers.facebook.com и внести ключи этого приложения в настройки сайта.

Ниже представлена подробная инструкция по созданию приложения и его настройке. Скриншоты к данной инструкции расположены ниже и пронумерованы.

После создания приложения, необходимо отметить пункт "Инстаграм" и сохранить изменения.

### Инструкция

1. Аккаунт разработчика Facebook:
   1. Необходимо иметь аккаунт на facebook.com и использовать его для входа на [https://developers.facebook.com/](https://developers.facebook.com/). Если у вас нет аккаунта — создайте его.
2. Следовать инструкциям [https://developers.facebook.com/docs/instagram-basic-display-api/getting-started](https://developers.facebook.com/docs/instagram-basic-display-api/getting-started):
   1. Создать приложение Facebook;
   2. Настройки → Основное → прокрутите до конца страницы и в самом низу нажмите "Добавить платформу" → Веб-сайт → Указать сайт → Сохранить изменения;
   3. Нажмите "Продукты (Товары)" на знак «+», найдите карточку Instagram Basic Display и нажмите на кнопку "Настроить";
   4. Выберите "Базовое представление (Basic Display)" (слева в меню Instagram Basic Display), прокрутите до конца страницы и нажмите "Создать приложение (Create New App)";
   5. Укажите название приложения Facebook, которое вы только что создали (ID приложения введённое ранее);
   6. Теперь на этой странице прокрутите немного ниже и пропишите настройки приложения:
      1. **Клиентские настройки OAuth**\
         Действительные URL переадресации для Oauth — адрес вашего сайта + /api/index/instagram (Пример: https://demo.wow2print.com/api/index/instagram);
      2. **Отменить авторизацию**\
         Деавторизация URL обратного вызова — адрес вашего сайта (https://demo.wow2print.com/);
      3. **Запросы на удаление данных**\
         URL запроса на удаление данных — адрес вашего сайта (https://demo.wow2print.com/);
      4. Сохранить (в правом нижнем углу окна браузера – Сохранить изменения).
3. Добавьте тестового пользователя Instagram:
   1. Выберите "Роли" (слева) → Роли и прокрутите до раздела "**Тестировщики Instagram**". Нажмите "Добавить тестировщиков Instagram (Instagram Testers)", введите ваше имя пользователя Instagram и отправьте приглашение;
   2. Откройте новое окно браузера, перейдите на сайт [www.instagram.com](https://www.instagram.com/) и войдите в приглашенный аккаунт Instagram. Выберите Значок профиля → "Редактировать профиль" → "Приложения и сайты" (слева 3 пункт) → "Приглашения тестировщиков" и примите приглашение.

{% hint style="warning" %}
Идентификатор приложения и Ключ приложения вводим заполняем в настройках интеграции с Instagram (Админ-панель → Настройки → Интеграции → Instagram
{% endhint %}

Далее переходим к Подверждению приложения, для этого необходимо составить заявку (после подтверждения вы получите доступ к Профилю/Контенту профиля клиента):

1. Для этого переходим в приложение instagram → Раздел "Товары" (левое меню) → Раскрываем список Instagram Basic Display → "Basic Display";\
   **Скриншот №1**
   1. На странице находим блок "Проверка приложения Instagram Basic Display" → Сразу нажимаем на оба варианта "Добавить к заявке";
   2. Появится Блок "Текущая заявка"\
      **Сриншот №2**
2. При нажатии на "Редактировать информацию" в обоих случаях выходит одинаковое окно, заполняем его так же одинаково:\
   **Скриншот №3**
   1. В блок "Подробно опишите…" необходимо заполнить то, каким образом будет использоваться данное приложение на вашем сайте. Пример текста:
      1. Перейти на сайт https://demo2.wow2print.com;
      2. Выбрать в меню "Продукция" → "Фотопродукция" → "Стандартные фотографии";
      3. Нажать кнопку "Заказать";
      4. В появившемся окне нажать на вкладку Instagram;
      5. В появившемся окне ввести логин/пароль от учетной записи Instagram;
      6. Далее выбрать "Сохранять" или "Не сохранять данные для входа";
      7. Нажать кнопку "Продолжить";
      8. Выбрать фотографии и нажать на кнопку "Вставить";
      9. Разместить фотографию на шаблоне (путем клика или переноса фотографии на шаблон);
      10. Нажать на кнопку "Добавить в корзину";
      11. Нажать на кнопку "Перейти в корзину";
      12. Выбираем способ доставки;
      13. Заполняем данные получателя;
      14. Выбираем способ оплаты;
      15. Нажать на кнопку "Оформить заказ";
      16. Оформленный заказ отправится в печать → в установленные сроки доставки клиент получит напечатанные фотографии из Instagram.
   2. В блок "Загрузки файла" необходимо загрузить Видеоинструкцию с проделанными шагами (описанными выше);
   3. После заполнения всех полей необходимо «Отпарвить» заявку.
3. Следом необходимо нажать на «Редактировать» напротив Подтверждения приложения:\
   **Скриншот №4**
   1. Проставить галочку возле «Подтверждаю, что все обязательные шаги выполнены»;
   2. Сохранить.
4. Так же необходимо просмотреть и выполнить все необходимые действия описанные снизу страницы;
5. В самом низу страницы «Сохранить изменения»;\
   **Скриншот №5**
6. После всех проделанных шагов, необходимо дождаться ответа от facebook (от 1 до 7 дней);
7. После подтверждения приложения необходимо перейти к подтверждению вашей компании, для этого необходимо нажать на «Перейти к подтверждению» и следовать инструкции.\
   **Скриншот №6**
