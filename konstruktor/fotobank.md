# Фотобанк

## **Основная информация**

Фотобанк представляет собой хранилище альбомов с изображениями, которые клиенты могут использовать при создании проектов в редакторе полиграфии.

При нажатии на кнопку загрузки фото клиенту предлагается два варианта:

1. Загрузить изображение со своего компьютера
2. Выбрать фото из фотобанка (готовые изображения, предоставленные типографией)

После выбора нужного альбома и конкретного изображения, оно загружается в соответствующую ячейку проекта.

#### **Интерфейс раздела фотобанка:**

В интерфейсе фотобанка представлен полный перечень доступных альбомов. Пользователи могут организовать этот список, отсортировав альбомы по названию, или быстро найти нужный через поисковую строку. Для каждого альбома в списке отображается дополнительная информация: справа от названия указывается количество содержащихся изображений в поле "Кол-во изображений", а также перечисляются Продукты, где данный альбом из фотобанка доступен для использования.

## **Создание нового альбома**

1. Нажмите кнопку **"Создать альбом"** в правом верхнем углу
2.  В открывшемся меню:

    2.1. Введите название альбома

    2.2. Нажмите **"Сохранить"**

<figure><img src="../.gitbook/assets/image (3920).png" alt=""><figcaption></figcaption></figure>

## **Редактирование альбома**

После создания альбома открывается страница редактирования с двумя вкладками:

### **Вкладка Описание**

* **Изменение названия**: отредактируйте название альбома при необходимости
* **Настройка продуктов**:
  1. Нажмите кнопку **"Изменить"** рядом с полем "Продукты"
  2. В открывшемся меню отметьте галочками продукты, где должен отображаться альбом
  3. Подтвердите выбор

{% hint style="info" %}
Если не отмечен ни один продукт, альбом будет доступен во всех продуктах
{% endhint %}

* **Удаление продукта**: нажмите на крестик рядом с названием продукта

<figure><img src="../.gitbook/assets/image (3921).png" alt=""><figcaption></figcaption></figure>

### **Вкладка Изображения**

Что бы добавить сюда изображение, нажмите кнопку **"Загрузить",** и выберите изображение с вашего устройства

#### **Требования к изображениям:**

✔ Минимальный размер: 643×300 пикселей\
✔ Поддерживаемые форматы: JPG, JPEG, PNG, WEBP, SVG

* **Удаление фото**: нажмите на значок корзины под соответствующим изображением

<figure><img src="../.gitbook/assets/image (3922).png" alt=""><figcaption></figcaption></figure>

## **Управление альбомами**

#### **Редактирование альбома:**

* Кликните на название нужного альбома в общем списке

#### **Управление видимостью:**

* Используйте тумблер для включения/отключения отображения альбома в редакторе полиграфии

#### **Удаление альбома:**

1. Найдите нужный альбом в общем списке
2. Нажмите на значок корзины в конце строки с названием альбома
3. Подтвердите удаление
