# Шрифты

В данном разделе представлен полный список шрифтов, доступных для использования во всех "Онлайн-редакторах" вашего сайта, включая: редакторы фотокниг, фотокарточек, фотокалендарей и  редактор полиграфии.

<figure><img src="../.gitbook/assets/image (150).png" alt=""><figcaption></figcaption></figure>

## **Структура раздела**

* Слева находятся названия шрифтов. Список можно отсортировать по алфавиту или найти по названию нужный шрифт.
* По центру расположен пример текста, который показывает, как выглядит шрифт.
* Справа в строке тумблер для включения или отключения шрифта во всех редакторах.

{% hint style="info" %}
**Важно:** Вы можете добавить не более **10 собственных шрифтов**.
{% endhint %}

<figure><img src="../.gitbook/assets/image (4) (1).png" alt=""><figcaption></figcaption></figure>







## Как добавить новый шрифт <a href="#kak-dobavit-novyi-shrift" id="kak-dobavit-novyi-shrift"></a>



1. Нажмите кнопку **«Добавить шрифт»** в правом верхнем углу экрана.
2. Вы перейдёте на страницу загрузки шрифта.
3. В поле **«Шрифт»** обязательно укажите имя. Без него сохранение будет невозможно.
4. Ниже расположены 4 блока для загрузки стилей текста:
   * **Обычный стиль (обязательно)**
   * Жирный стиль
   * Курсивный стиль
   * Жирный курсив

<figure><img src="../.gitbook/assets/image (4090).png" alt=""><figcaption></figcaption></figure>







### **Требования к загрузке шрифтов**

{% hint style="warning" %}
* **Форматы:** `.ttf`
* **Обязательное условие:** «Обычный стиль» должно быть загружен обязательно.
{% endhint %}



#### **Как загрузить файл:**

1. Нажмите кнопку **«Выберите файл»** в нужном стиле.
2. Найдите файл шрифта на вашем компьютере.
3. После успешной загрузки справа от кнопки появится название загруженного файла.
4. Нажмите кнопку «Сохранить».

<figure><img src="../.gitbook/assets/image (4091).png" alt=""><figcaption></figcaption></figure>







## **Управление загруженными шрифтами**

Любой добавленный вами шрифт (кроме стандартных) можно удалить или изменить.

#### _Удалить:_

1. Найдите нужный шрифт в списке.
2. Справа в строке шрифта нажмите кнопку с иконкой **корзины**. ![](<../.gitbook/assets/image (4094).png>)
3. Подтвердите действие. Шрифт будет удалён из списка.&#x20;

#### _Изменить:_

1. Найдите нужный шрифт в списке.
2. Справа в строке шрифта нажмите кнопку с иконкой **карандаша**. ![](<../.gitbook/assets/image (4095).png>)
3. Откроется страница редактирования шрифта.

<figure><img src="../.gitbook/assets/image (4093).png" alt=""><figcaption></figcaption></figure>





* Название загруженного файла отображается под строкой соответствующего стиля начертания.
* **Заменить файл** можно в любой момент — просто загрузите новый файл в том же стиле.
* **Удалить стиль**: после сохранения у всех стилей, кроме **Обычного стиля**, появляется кнопка **«Удалить стиль»**.
