# Скидочный баннер

Перейдите в Маркетинг - Скидочный баннер и нажмите кнопку "Создать"

<figure><img src="../.gitbook/assets/Screenshot_1 (18).png" alt=""><figcaption></figcaption></figure>



Заполните все поля:

* _Название_&#x20;
* _Скидка -_ выберите скидку из списка
* _Вид -_ как будет располагаться скидочный баннер на сайте.&#x20;
* _Поведенческий фактор -_ **Без выбора**; **Просмотр уникальных страниц** (необходимо будет вставить URL страницы на сайте без указания домена, баннер будет появляться только на этой странице) или **Количество просмотров уникальных страниц** (после просмотра посетителем сайта указанного количества страниц, баннер станет виден на сайте)
* _Тайм-аут (секунды) -_ время, после которого баннер станет виден на странице
* _Цвет баннера_
* _Отображать на баннере -_ **Список продукции** (на которые распространяется скидка) или **Кнопка** (появятся дополнительные поля "Подпись кнопки" и "Ссылка кнопки")

<figure><img src="../.gitbook/assets/Screenshot_2 (15).png" alt=""><figcaption></figcaption></figure>



Не забудьте включить отображение баннера

<figure><img src="../.gitbook/assets/Screenshot_10 (4).png" alt=""><figcaption></figcaption></figure>



{% hint style="info" %}
**Важно!**

**Возможно создать только один скидочный баннер на одну скидку.**

**На странице может отображаться только один баннер**
{% endhint %}

Если скидка действует для конкретных пользователей или отдельных групп клиентов, скидочный баннер будет отображаться на сайте только для этих пользователей и только после их авторизации на сайте.
