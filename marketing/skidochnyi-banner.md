# Скидочный баннер

Перейдите в Маркетинг - Скидочный баннер и нажмите кнопку "Создать"

<figure>

![](<../.gitbook/assets/Screenshot_1 (18).png>)

<figcaption>



</figcaption>

</figure>



Заполните все поля:

-  *Название* 

-  *Скидка -* выберите скидку из списка

-  *Вид -* как будет располагаться скидочный баннер на сайте. 

-  *Поведенческий фактор -* **Без выбора**; **Просмотр уникальных страниц** (необходимо будет вставить URL страницы на сайте без указания домена, баннер будет появляться только на этой странице) или **Количество просмотров уникальных страниц** (после просмотра посетителем сайта указанного количества страниц, баннер станет виден на сайте)

-  *Тайм-аут (секунды) -* время, после которого баннер станет виден на странице

-  *Цвет баннера*

-  *Отображать на баннере -* **Список продукции** (на которые распространяется скидка) или **Кнопка** (появятся дополнительные поля "Подпись кнопки" и "Ссылка кнопки")

<figure>

![](<../.gitbook/assets/Screenshot_2 (15).png>)

<figcaption>



</figcaption>

</figure>



Не забудьте включить отображение баннера

<figure>

![](<../.gitbook/assets/Screenshot_10 (4).png>)

<figcaption>



</figcaption>

</figure>



:::note 

**Важно!**

**Возможно создать только один скидочный баннер на одну скидку.**

**На странице может отображаться только один баннер**

:::



Если скидка действует для конкретных пользователей или отдельных групп клиентов, скидочный баннер будет отображаться на сайте только для этих пользователей и только после их авторизации на сайте.
